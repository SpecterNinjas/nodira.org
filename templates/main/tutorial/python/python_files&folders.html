Files & Folders I/O



Section 1.1: Files & Folders Overview

In this section, we'll learn about Python files & folders. Basically, when it comes to storing, reading, changing and deleting data, python provides us really useful functions and operators.

Mode Description
'r' reading mode (by default). It allows you only to read the ﬁle, not to modify it. When using this mode the ﬁle must exist.
'w' writing mode. It will create a new ﬁle if it does not exist, otherwise will erase the ﬁle and allow you to write to it.
'a' append mode. It will write data to the end of the ﬁle. It does not erase the ﬁle, and the ﬁle must exist for this mode.
'b' binary mode. It allows to open file in binary format.
'+' updating mode. It allows to open/read and do some modification.
'x' open for exclusive creation, will raise FileExistsError if the ﬁle already exists.
Section 2.1: Reading Files

  • Safely opening file with read mode using try/except block.

try :
    with   open (  "file.txt"   ,   "r"  )  as myfile  :
          #     Some operation with file
except   FileExistsError:
    #    Error message goes here

Section 2.2: Reading Files Line-By-Line

  • The simple logical way is using for loop.

try :
    with   open (  "file.txt"   ,   "r"  )  as myfile :
        for   line   in  myfile :
            print ( line )
except   FileExistsError:
    #    Error message goes here

  • Using built-in readline( ) :

try :
    with   open (  "file.txt"   ,   "r"  )   as   myfile :
        while  True :
            inline  =  myfile.readline( )
            if  inline ==    ' ' : #    When it comes to the end
                break
            print ( inline )
except   FileExistsError:
    #    Error message goes here

Section 2.3: Reading Full Contents of Files

  • For safety, it's recommended to use try/except block:

try :
    with   open (  "file.txt"   )   as   myfile :
        full_content =  myfile. read( )
        print ( full_content )
except   FileExistsError:
    #    Error message goes here

  • If you don't want to use with/as method, then you must close( ) the file manually:

try :
    myfile  =  open (  "file.txt" , "r"  ) :
    full_content =  myfile. read( )
    print ( full_content )
    myfile.close( ) #    Closing file manually
except   FileExistsError:
    #    Error message goes here

Section 3.1: Writing to a file

  • Since we're using with/as block, we don't need to worry about closing our file.

  • To write something into the file, we use write( ) function.

with   open (  "file.txt" , "w"  )   as   myfile :
    myfile.write ( "Hello" )
    myfile.write ( "World" )
    myfile.write ( "What's up?" )

  • Remember that Python will not automatically add line breaks.

#   "myfile.txt" :
HelloWorldWhat's up?

  • We must manually add line breaks using \n.

with   open (  "file.txt" , "w"  )   as   myfile :
    myfile.write ( "Hello \n" )
    myfile.write ( "World \n" )
    myfile.write ( "What's up? \n" )

#   "myfile.txt" :
Hello
World
What's up?

Section 4.1: Checking File/Path Existence

  • Checking file existence using os.path module :

import os.path
print ( os.path.isfile ( "mypath/myfile.txt" ) )

  • Checking path existence using os.path module :

import os.path
print ( os.path.exists ( "mypath" ) )

Section 5.1: Deleting File/Path

In order to delete file, we're using os.remove( ) function.

import os
print ( os.remove ( " myfile.txt" ) )

More safe way to delete a file is checking for existence first, using os.path.exists( ) or os.path.isfile( ) function. Section 4.1

import os.path
print ( os.path.exists ( "mypath/myfile.txt" ) )

In order to delete folder/path, we're using os.rmdir( ) function which stands for remove directory.

import os
print ( os.rmdir ( " myfolder" ) )

  • By running these command, the whole folder will be deleted.
