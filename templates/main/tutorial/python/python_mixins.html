Python Mixins



Section 1.1: Python Mixin Overview

A Mixin is a special kind of functionality or properties that can be used in various classes, but don't come from a base class.

It's very similar to inheritance, where same functionality or property is inherited from base class.

Section 1.3: Python Mixin Conclusion

A mixin can basically be thought of as a set of code or functionality that can be added to one or more classes to add additional capabilities without using inheritance.

In Python, a mixin is code wrapped up in a module that a class can include or extend. In fact, a single class can have many mixins.
