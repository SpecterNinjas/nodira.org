Python *args and **kwargs



Section 1.1: Python *args Overview

The * prefix is used to grab unnamed arguments where there's no any information about the number of numbers.

The * prefix is usually used with variable args, but it's up to you.

Section 1.2: *args in Practice

  • * prefix will catch all unpacked numbers:

#    * prefix will catch all unpacked numbers
first , *rest  =  [ 1 , 2 , 3 , 4 , 5 ] #    first  = 1 , rest  = [ 2 , 3 , 4 , 5 ]
first , second , *rest  =  [ 1 , 2 , 3 , 4 , 5 ] #    first  = 1 , second  = 2 , rest  = [ 3 , 4 , 5 ]
first , *middle , last  =  [ 1 , 2 , 3 , 4 , 5 ] #    first  = 1 , middle  = [ 2 , 3 , 4 ] , last  = 5
*rest , last  =  [ 1 , 2 , 3 , 4 , 5 ] #    rest  = [ 1 , 2 , 3 , 4 ] , last  = 5

  • Usage of *args within a function:

def  numbers ( first , *args ) :
    print ( "First arg: " ,  first )
    print ( "The rest: " ,  *args )

numbers ( 1 , 2 , 3 , 4 , 5 )
#    First arg:  1
#    The rest:  2   3   4   5

  • Applying * prefix operator with zip ( ) function:

a  = [ 1 , 2 , 3 , 4 , 5 ]
b  = [ 6 , 7 , 8 , 9 , 10 ]
#    Simple usage of zip ( ) function:
zipped  =  zip ( a , b )
print ( *zipped ) #    ( 1 , 6 ) ( 2 , 7 ) ( 3 , 8 ) ( 4 , 9 ) ( 5 , 10 )



Section 2.1: Python **kwargs Overview

The ** prefix is used to grab arbitrary number of named arguments.

The ** prefix is usually used with variable kwargs, but it's up to you.

Section 2.2: **kwargs in Practice

  • **kwargs keyword is used to catch an arbitrary number of named arguments:

def  numbers ( **kwargs ) :
    print ( kwargs )

numbers ( a  =  "one" , b  =  "two" , c  =  "three" )
#   Output:   { 'a' : 'one' , 'b' : 'two' , 'c' : 'three' }

  • **kwargs will only catch named arguments:

def  numbers ( first , **kwargs ) :
    print ( first )
    print ( kwargs )

numbers ( 5 , a  =  "one" , b  =  "two" , c  =  "three" )
#   Output:   5
#   Output:   { 'a' : 'one' , 'b' : 'two' , 'c' : 'three' }

  • Splitting keyword and value:

def  numbers ( **kwargs ) :
    for  key , value  in  kwargs.items (  ) :
        print ( f" Key: {key} and value: {value} "   )

numbers ( a  =  "one" , b  =  "two" , c  =  "three" )
#   Output:   Key: a and value: one
#   Output:   Key: b and value: two
#   Output:   Key: c and value: three
