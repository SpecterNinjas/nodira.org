Python Map & Reduce Function



Section 1.1: Python Map Overview

The map( ) function simply applies a set of specified function to each and every element of an iterable.

Section 1.2: Python Map Syntax

  • The syntax of map( ) function:

map ( function , iterables )

Parameter Description
function Required function for mapping. Applies a set of function to each item of an iterable.
iterable iterable in which function executes a set of function ( multiple iterables allowed ).
Section 1.3: Map Function Practice

  • Note that result needs to be converted to list , tuple , set and so on.

my_iter  =    [  "Hello" , "World" , "Python" , "Map" , "Function"  ]
#   applies len function to each item of an iterable
print ( map ( len , my_iter ) ) #   'map object at 0x7f2ebc495370'
list ( map ( len , my_iter ) ) #   [ 5 , 5 , 6 , 3 , 8 ]

  • We used lambda inline function to define our function. Learn More

my_iter  =    [  1 , 2 , 3 , 4 , 5  ]
#   applies len function to each item of an iterable
list ( map ( lambda x : x**2 , my_iter ) ) #   [ 1 , 4 , 9 , 16 , 25 ]
list ( map ( lambda x : x + 1 , my_iter ) ) #   [ 2 , 3 , 4 , 5 , 6 ]



Section 2.1: Python Reduce Overview

The reduce ( ) function is used to reduce the iterable via set of specified function.

Section 2.2: Reduce Function Syntax

  • The syntax of reduce( ) function:

reduce ( function , iterable , initializer )

Parameter Description
function Required function to reduce iterables. Applies a set of operation to over the iterable ( 2 arguments required ).
iterable Iterable in which function executes a set of function.
initializer Starting point of the reduction.
Section 2.3: Reduce Function Practice

  • Note that functools library needs to be imported in order to use reduce function.

from   functools   import   reduce
my_iter  =    [  1 , 2 , 3 , 4 , 5  ]
#   Calculates the sum of all items
print ( reduce ( lambda a , b : a + b , my_iter ) ) #   15
#    Calculates the sum of all items with initial value of 20.
print ( reduce ( lambda a , b : a + b , my_iter , 20) ) #   35

  • We used lambda inline function to define our function. Learn More

from   functools   import   reduce
my_iter  =    [  1 , -2 , 3 , 4 ]
#   Calculates the Multiplication of all items
print ( reduce ( lambda a , b : a * b , my_iter ) ) #   -24
#    Calculates the Multiplication of all items with initial value of 20.
print ( reduce ( lambda a , b : a * b , my_iter , 20) ) #   -480
