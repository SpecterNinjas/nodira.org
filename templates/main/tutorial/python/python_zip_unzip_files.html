Python Zip & Unzip Files



Section 1.1: ZIP Files Format Overview

The ZIP file format is a common archive and compression standard.

The ZIP Module handles ZIP files that use only the ZIP64 extensions.

Hence, it supports decryption of encrypted files in ZIP archives, but it currently cannot create an encrypted file.

Section 2.1: Python ZipFile Objects

  • The syntax of ZipFile( ) function:

ZipFile ( file , mode  =  'r', compression  =  ZIP_STORED, allowZip64  =  True )

Parameter Description
file File that needs to be zipped or it can be a path to a file.
mode Mode that file that needs to be zipped.
compression ZIP compression method to use when writing the archive.
allowZip64 Zipfile will create ZIP files that use the ZIP64 extensions when the zipfile is larger than 4 GB.

  • Creating a zip archive  of a directory in Python :

import   os
import   zipfile #   importing zipfile

def    zipdir ( path , ziph ) :
    for    root , dirs , files   in   os.walk ( path ) :
        for    file   in   files :
            ziph.write ( os.path.join ( root , file ) )

zipf = zipfile.ZipFile ( 'Python.zip' , 'w' , zipfile.ZIP_DEFLATED )
zipdir ( 'tmp/' , zipf)
zipf.close (  )

Section 2.2: ZIP File Contents

The printdir (  ) is a function to inspect contents of a zip archive.

Let's apply this function in practice :

from   zipfile   import   Zipfile #   importing Zipfile

with   Zipfile ( file )   as   zip :
    zip.printdir (  )
#   Output:
#   Filename   Modified Size
#   index.py   2019-11-15 20:22:14   341215
#   style.css   2019-10-08 20:22:14   23315
#   cards.py   2017-10-14 20:22:14   151676
#   sidebar.js 2018-08-25 20:22:14   7639

  • The namelist (  ) is a function to print a list of filenames of a zip archive :

from   zipfile   import   Zipfile #   importing Zipfile

with   Zipfile ( file )   as   zip :
    print ( zip.namelist (  ) )
#   Output:     [ 'index.py' , 'style.css' , 'cards.py' , 'sidebar.js' ]

Section 3.1: Unzip - Opening ZIP Files

The extractall (  ) function is used to unzip all files.

Let's apply this function in practice :

from   zipfile   import   Zipfile #   importing Zipfile

with   Zipfile ( "archive.zip" , 'r' )   as   zip :
    zip.extractall (  ) #   extracting all files in current directory
    zip.extractall ( "path") #   extracting all files in specified directory

The extract (  ) function is used to unzip a single file.

from   zipfile   import   Zipfile #   importing Zipfile

f  =  open ( 'archive.zip' , 'rb' )
archive  =  Zipfile ( f )

for   file   in   archive.namelist (  ) :
    archive.extract ( file , path ) #   extracting all files one-by-one

Section 3.2: Add Files To ZIP Archives

Multiple files can be added to ZIP Archive :

from   zipfile   import   Zipfile #   importing Zipfile
write_mode  =  Zipfile ( 'archive.zip' , 'w' )

Files that needs to be added :

write_mode.write ( "main.txt" , "style.js" )
write_mode.close (   )

For adding string , writestr (  ) method is used :

my_str  =  "Some text here!"
write_mode.writestr ( "file.txt" , my_str )
write_mode.close (   )
