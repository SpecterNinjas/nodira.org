Python Overriding Methods



Section 1.1: Python Overriding Overview

This section focuses on Overriding methods and how to implement in Python projects.

Method overriding is a feature that allows a subclass or child class to provide a specific implementation of a method that is already provided by one of its superclasses or parent classes.

Let's do some practice and learn about overriding methods on a deep level.

Section 2.1: Python Overriding Practice - Part 1

Let's create a parent class Factory. Then a class that inherits from the class Farm.
We add a method employee that we override:

class   Factory :
    def   employee ( self ) :
        print ( "I work at Factory" )

class   Farm ( Factory ) :#     inheriting Factory class
    def   employee ( self ) :
        print ( "I work at Farm" )#     method has been overridden

a  =  Farm (  )
a.employee (  ) #     I work at Farm

Section 2.2: Python Overriding Practice - Part 2

In order to understand Overriding Methods on a deep level, let's do one more exercise:

class   Factory :
    def   employee ( self ) :
        print ( "I work at Factory" )

class   Farm ( Factory ) :#     inheriting Factory class
    def   employee ( self ) :
        print ( "I work at Farm" )#     method has been overridden

class   Branch ( Factory ) :#     inheriting Factory class
    def   employee ( self ) :
        print ( "I work at Branch" )#     method has been overridden

a  =  Farm (  )
b  =  Branch (  )

a.employee (  ) #     I work at Farm
b.employee (  ) #     I work at Branch

Section 2.3: Overriding Conclusion

Method overriding - when a class is inherited,  you can override the definition of any of its existing accessible methods in its subclass by a feature known as method overriding.

In simple words, method overriding is changing the definition of a superclass method in its subclass.
