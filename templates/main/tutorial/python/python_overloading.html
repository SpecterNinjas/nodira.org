Python Overloading Methods



Section 1.1: Python Overloading Overview

This section focuses on Overloading method and how to implement in Python projects.

Overloading is a feature that allows a class to have more than one method having the same name, if their argument lists are different.

Let's do some practice and learn about overloading on a deep level.

Section 2.1: Python Overloading Practice - Part 1
Method Overloading: Changing number of arguments

Let's create 2 functions with the same name and see what happens :

#   Simple example of Overloading

#   this function takes 2 arguments
def   product ( a , b ) :
    output  =  a * b
    print ( output )

#   this function takes 3 arguments
def   product ( a , b , c ) :
    output  =  a * b * c
    print ( output )

product ( 3 , 4 ) #   12   ( 1st function is executed )
product ( 4 , 5 , 6 )#   120   ( 2nd function is executed )
product ( 4 , 5 , 6 , 7 )#   TypeError :   product (  ) takes 3 positional arguments but 4 were given

Section 2.2: Python Overloading Practice - Part 2
Method Overloading: Changing data type of arguments

pip3   install   multipledispatch#    Installing Multiple Dispatch Decorator

Let's create several functions with the same name , but different arguments :

from   multipledispatch   import   dispatch

#   Function invokes when takes 2 integer numbers
@dispatch ( int , int )
def   product ( a , b ) :
    output  =  a * b
    print ( output )

#   Function invokes when takes 2 floating point numbers
@dispatch ( float , float )
def   product ( a , b ) :
    output  =  a * b
    print ( output )

#   Function invokes when takes 1st integer number , 2nd float number
@dispatch ( int , float )
def   product ( a , b ) :
    output  =  a * b
    print ( output )

#   Function invokes when takes 1st float number , 2nd integer number
@dispatch ( float , int )
def   product ( a , b ) :
    output  =  a * b
    print ( output )

product ( 3 , 4 ) #   12   ( 1st function is executed )
product ( 3.5 , 4.5 )#   15.75   ( 2nd function is executed )
product ( 3 , 4.5 )#   13.5   ( 3rd function is executed )
product ( 3.5 , 4 )#   14   ( 4th function is executed )
product ( 3 , 4 , 5 )#   TypeError :   product (  ) takes 2 positional arguments but 3 were given

Section 2.3: Overloading Conclusion

Overloading is the class having methods that are the same name with different arguments. Arguments different will be based on a number of arguments and types of arguments. It is used in a single class. It is also used to write the code clarity as well as reduce complexity.

Advantages of Overloading

  • It supports readability making your code ( classes, methods ) reusable and clean.

  • Flexibility to call a similar method for different types of data.

  • It is used on constructors to create new objects given different amounts of data.

Disadvantages of Overloading

  • Waste of time to write and maintain thousands lines of code.

  • Hard to maintain, whereas in new scenarios you need write more and more new functions.
