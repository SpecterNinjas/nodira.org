Python Dictionary



Section 1.1: Python Data Type

In this section, we'll cover : Python Dictionary

  • Dictionary - a collection of unordered elements.

  • Dictionary - mutable (meaning that we can change its values).

  • Dictionary - supports key-based indexing.

  • Dictionary - every key must be unique.

  • Dictionary - curly brackets{ } are used, like sets.

Section 1.2: Dictionary Example

Dictionary elements are separated by commas ( like sets ) , and closed with curly brackets.

  • dict { "key" : "value" }

b  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
print ( b )#    'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy'

In this example, keys : 'one' , 'two' , 'three' and values : 'dog' , 'cat' , 'puppy', respectively.

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
a.keys (  )#    dict_keys ( [ 'one' , 'two' , 'three' ] )

.keys ( ) function returns key attributes of a dictionary.

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
a.values (  )#    dict_values ( [ 'dog' , 'cat' , 'puppy' ] )

.values ( ) function returns value attributes of a dictionary.

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
a.items (  )#    { ( 'one' : 'dog' ) , ( 'two' : 'cat' ) , ( 'three' : 'puppy' ) }

.items ( ) function returns paired ( key , value ) attributes of a dictionary.

Section 1.3: Accessing Dictionary items

1st way : Dictionary items can be accessed through for loop. We'll learn more detailed later.

dict  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
for   key , value    in   dict.items (  ) :
      print ( dict[key] )#    Output : dog , cat , puppy


2nd way : using key-based indexing.

b  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
b['one']#    Output: dog

b  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
b['two']#    Output : cat

b  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
b['dog'] #    Error : use only keys for indexing.

3rd way : get ( ) function.

b  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
b.get ( 'one' )#    Output : dog
b.get ( 'four' , 'default_value' )#    Output : default_value

  • Note that get ( ) function returns 2nd argument, in case specified key doesn't exist.

Section 1.4: Combining 2 dictionaries

  • merged_dict  =  { **dict1 , **dict2 }

a  =  { 'one' : 'dog' , 'two' : 'cat' }
b  =  { 'word' : 'hello' }
c  =  { **a , **b }#   merging a and b.
print ( c )#   Output: 'one' : 'dog' , 'two' : 'cat' , 'word' : 'hello'

Section 1.5: Adding elements in a dictionary

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
a [ 'four' ]  =  'new_element'
print ( a ) #   'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' , 'four' : 'new_element'

Section 1.6: Checking Key Existence

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
'four'   in   a. keys (  )#   False

Section 1.7: Checking Value Existence

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
'cat'  in   a.values ( )#   True

Section 2.1: Built-in functions

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
a.pop ( 'one' )#   'two' : 'cat' , 'three' : 'puppy'  

  • pop(key) function removes specified key with its paired value.

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
len ( a )#   3  

  • len(dict) returns length of a dictionary.

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
a.popitem (  )#   'one' : 'dog' , 'two' : 'cat'  

  • popitem( ) removes last added item.

a  =  { 'one' : 'dog' , 'two' : 'cat' , 'three' : 'puppy' }
a.clear (  )#   empty  

  • clear( ) clears all paired items.
