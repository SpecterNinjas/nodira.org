Python Data & Time



Section 1.1: Python datetime package

Basically, Python offers already built-in datetime package to deal with all problems related to date and time.

from datetime import datetime   #     importing datetime module

now  =  datetime.now (  )#     2020-05-19    18:49:10.920332
today  =  datetime.today (  )#     2020-05-19    18:49:10.920332
year  =  datetime.now (  ).year#     2020
month  =  datetime.now (  ).month# 5
day  =  datetime.now (  ).day#     19
combination  =  f'{month}:{day}:{year}'#     05:19:2020
hour  =  datetime.now (  ).hour#     18
minute  =  datetime.now (  ).minute#     49
second  =  datetime.now (  ).second#     10
mixture  =  f'{hour}-{minute}-{second}'#     18-49-10

Section 1.2: Set date&time   

We'll set desired date and time :   datetime ( year , month , day )

from datetime import datetime
set_datetime  =  datetime ( 2015 , 8 , 22 )#     set datetime :   22th of August, 2015

Section 1.3: Calculating date&time difference

Here, we will calculate our time difference :

Subtraction method

from datetime import datetime #   import datetime module
now  =  datetime.now (  )#   2020-05-19    19:05:14.927861
future  =  datetime ( 2022 , 9 , 16 )#   set datetime :   16th of September, 2022
difference  =  future  -  now#   849 days, 4:54:45.072139

Section 1.4: Manipulation with timedelta

Add/Subtract desired date and time :   timedelta ( years, weeks, days, . . . , seconds )

from datetime import datetime, timedelta
future_datetime  =  datetime.now (  ) + timedelta ( days  =  4 , hours  =  5 , seconds  =  3 )#    Add 4 days, 5 hours and 3 secs
past_datetime  =  datetime.now (  )  -  timedelta ( days  =  4 , hours  =  5 , seconds  =  3 )#    Subtract 4 days, 5 hours and 3 secs

Section 1.5: Fuzzy Attribute

In order to convert text to numerical date :   fuzzy = True

from dateutil.parser import parse
text  =  parse ( " My birthday is January 22 , 1999 at 3:36:05AM ", fuzzy  =  True )#    Converting text to numerical date
print ( text )#    1999 , 1 , 22 , 3 , 36 , 5

Section 1.6: Tomorrow vs Yesterday

Usage of  timedelta ( years, weeks, days, . . . , seconds )

from datetime import datetime , timedelta
tomorrow  =  datetime.now (  )  +  timedelta ( days  =  1 )#    yesterday
today  =  datetime.now (  )#    today
yesterday  =  datetime.now (  )  -  timedelta ( days  =  1 )#    tomorrow

Section 1.7: Datetime Output Formats

Usage of  strftime ( %a  %A  %b  %B . . . )

from datetime import datetime
now  =  datetime.now (  )#    2020-05-19    21:43:39.278697
format1  =  now.strftime ( "%d  %B  %Y" )#    19 May 2020
