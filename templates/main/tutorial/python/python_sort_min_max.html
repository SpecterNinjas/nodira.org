Python Sorting & Min-Max Function



Section 1.1: Python Sorting Overview

The sorted ( ) function is used to sort elements of an iterable. Sorting can be defined either in ascending or descending order.

In sorted ( ) function, strings are sorted in alphabetical and numbers are sorted in numerical order.

Section 1.2: Python Sorted Syntax

  • The syntax of sorted( ) function:

sorted ( iterable , key = None , reverse = False )

Parameter Description
iterable Required parameter for sorting. A list of elements/iterables that can be list, tuple, set and so on.
key Key function for sorting comparison. Default is None. (Optional)
reverse can be either True or False. True for sorting in descending order, False for ascending order. Default is False. (Optional)
Section 1.3: Sorted Function Practice

  • Note that string and number mixed data types can't be sorted, Error occurs.

my_str  =    [  "H" , "W" , "P" , "M" , "A"  ]
#   strings will be sorted alphabetically ascending
print ( sorted ( my_str ) ) #   [ -15 , -2 , 5 , 13 , 21 , 34 ]
#   strings will be sorted alphabetically descending
print ( sorted ( my_str , reverse = True ) ) #   [ 34 , 21 , 13 , 5 , -2 , -15 ]

  • By default, sorted function will sort data in ascending order.

numbers  =    [  21 , -2 , 13 , 34 , -15 , 5  ]
#   numbers will be sorted numerically ascending
print ( sorted ( numbers ) ) #   [ 'A' , 'H' , 'M' , 'P' , 'W' ]
#   numbers will be sorted numerically descending
print ( sorted ( numbers , reverse = True ) ) #   [ 'W' , 'P' , 'M' , 'H' , 'A' ]


Section 2.1: Python Minimum & Maximum Overview

The min ( ) and max ( ) function is used to find minimum and maximum argument.

Section 2.2: Min & Max Function Syntax

  • The syntax of min( ) and max( ) function:

min ( iterable )

max ( iterable )

Parameter Description
iterable A list of data from which function will find minimum and maximum argument.
Section 2.3: Min & Max Function Practice

  • Note that string and number mixed data types can't be sorted, Error occurs.

my_str  =    [  "Hello" , "World" , "Python" , "Machine" , "Alexa"  ]
#   function  will  return  alphabetically  largest
print ( max ( my_str ) ) #   World
#   function  will  return  alphabetically  smallest
print ( min ( my_str ) ) #   Alexa

  • min ( ) and max ( ) function will compare these numbers numerically.

numbers  =    [  21 , -2 , 13 , 34 , -15 , 5  ]
#   numbers will be compared numerically
print ( max ( numbers ) ) #   34
print ( min ( numbers ) ) #   -15
