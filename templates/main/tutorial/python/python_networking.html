Python Networking



Section 1.1: Python Networking Overview

This section focuses on Python Networking and how to implement in Python projects.

Networking is a connection that allows at least 2 nodes to be linked with each other in order to exchange important files, share resources and build communication bridge among all connected nodes.

Let's do some practice and learn about Python Networking and how to implement & understand them on a deep level.

Section 2.1: Python Networking - Simple Http Server

Simple Http Server   is a built-in module of Python that's used to build simple connection over certain port number.

By default, Python provides port number 8000 to build simple connection.

>>> python3  -m  http.server  9090

Default port 8000 will be used in case you omit to specify port number, but in above example we told to our server to use 9090.

import   http.server
import   socketserver #   used to setup a network serve

PORT  =  8000

handler  =  http.server.SimpleHTTPRequestHandler
httpd  =  socketserver.TCPServer ( ( " " , PORT ) , handler )
print ( "Serving Port " , PORT )
httpd.serve_forever (  )

Section 3.1: Python Networking - TCP Server

Let's do some practice and start creating TCP server. Here, we created 2 python files : client.py and server.py

#    server.py
from    socketserver   import   ThreadingTCPServer , BaseRequestHandler

class   EchoHandler ( BaseRequestHandler ) :
    def   handle ( self ) :
        print ( 'Connected : ' , self.client_address )
        while   True :
            msg  =  self.request.recv ( 9090 )
            if   not   msg :
                break
            self.request.send ( msg )

if   __name__  ==  '__main__' :
    server  =  ThreadingTCPServer ( ( ' ' , 5000 ) , EchoHandler )
    server.serve_forever (  )

Now it's time to define our client.py file to handle user network connection :

#    client.py
from    socket   import   socket , AF_INET , SOCK_STREAM

sock  =  socket ( AF_INET , SOCK_STREAM )
sock.connect ( ( 'localhost' , 5000 ) )
sock.send ( b'Python is cool!' )
sock.recv ( 9090 ) #    b'Python is cool!'

Section 3.2: Python Networking - UDP Server

Now we are going much deeper and learn how to create UDP server.

UDP server creation is very similar to TCP Server, but contains some additional functions.
