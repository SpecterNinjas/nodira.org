Python PIP Package Manager



Section 1.1: PIP Package Manager Overview

pip is one of the most widely used packages in Python. Basically, it's Indexed Package Manager that's used to install any Python package.

In this section, we'll learn about pip and how install any packageon a deep level.

Section 1.2: Install Python Package

  • Installing the latest version of a package called PackageName:

>>>   pip install PackageName

  • Installing specific version of a package called PackageName:

>>>   pip install PackageName  ==  2.0.1

  • Installing a specified minimum version of a package called PackageName:

>>>   pip install PackageName  >=  2.0.1


Install From Requirements File:

  • Requirements File - a single .txt file in which each line will show you what kind of package needs to be installed.

>>>   pip install -r requirements.txt


Install All Files Of Requirements File:

  • Pip can be useful to create whole indicated packages in a Requirements File.

>>>   pip freeze > requirements.txt


Check All Installed Packages:

  • Freeze command will show all the packages that you ever installed.

>>>   pip freeze

Section 2.1: PIP All Packages

  • List command will also show all installed packages, like freeze.

>>>   pip list

#   Output:
behave ( 1.2.4 )
enum34 ( 1.0 )
flask ( 0.10.1 )

  • List Outdated command will show outdated packages along with new versions.

>>>   pip list --outdated

#   Output:
behave ( Current: 1.2.4 , Latest: 1.2.5 )
enum34 ( Current: 1.0 , Latest: 1.1)
flask ( Current: 0.10.1 , Latest: 0.10.2 )

Section 2.2: Upgrade Python Package

  • Upgrade with the latest version of a package called PackageName:

>>>   pip install --upgrade PackageName

  • Upgrade pip itself with the latest version pip :

>>>   pip install --upgrade pip

Section 2.3: Uninstall Python Package

  • Uninstall of a package called PackageName:

>>>   pip uninstall PackageName

Section 3.1: Update All Outdated Python Packages

  • Updating all packages in Windows:

for /F "delims= " %i in ('pip list --outdated --local') do pip install -U %i

  • Updating all packages in Linux:

pip list --outdated --local | grep -v '^\-e' | cut -d = -f 1 | xargs -n1 pip install -U
