Python User Input/Output



Section 1.1: User Input/Output Overview

In general, we use input function to get an input and print function to show the output of a user.

  • Note that in Python 2.x raw_input function was used to get an input :

Python 2.x:
hello  =  raw_input ( " What's up buddy ? " )
print ( " What's up buddy ? " + hello  )

Python 3.x:
hello  =  input ( " What's up buddy ? " )
print ( " What's up buddy ? " + hello  )

Section 1.2: Casting Errors

If you're new to Python, then you will have casting errors when you try to merge 2 different data types inside the print function.

  • print function is always string data type.

  • Here is some examples :

age  =  input ( " How old are you ? " )
print ( " How old are you ? " + age  )
next_year  =  age + 1 #    TypeError :  unsupported operand type(s) for 'str' and 'int'  

print ( " How old are you ? " + int (age) + 1  )

  • Literally, we can't combine str with other data types. In order to avoid error, we need to change other data types to str.

number  =  10.99
print ( " My number is : " + number  ) #    TypeError :  unsupported operand type(s) for 'str' and 'float'  
print ( " My number is : " + str ( number )  ) #    Output :  My number is :   10.99

  • Remember that input is always str data type.

score  =  input ( " What's your score from exam ? " )
print (  type ( score ) ) #    Output :  "str"

  • Converting str to float to do some mathematical operations.

score1  =  float ( input ( " What's your score from History ? " )  )
score2  =  float ( input ( " What's your score from Calculus ? " )  )
avg_score  =  0.5 * ( score1 + score2 )
print ( " My Average Score : " + str ( avg_score )  ) #   You can't merge 'str' with 'float', hence you need to convert 'float' to 'str'

Section 2.1: Applying format( ) function

str.format( ) is one of the widely used string formatting methods, through str.format( ) method you're able to insert multiple values (regardless of data type) within a string through positional formatting.

  • In str.format( ) method, braces and values are regarded respectively.

  • No space allowed inside {} (curly braces), otherwise KeyError raises.

number  =  10.99
my_str  =  "Hello World"
#    1st brace takes 1st value, 2nd brace takes 2nd value.
print ( " My number is {} and my string is {} ".format ( number , my_str )  )
#   Output:   My number is 10.99 and my string is Hello World

  • str.format( ) method can also be used by assigning variable names.

  • Variable name is defined inside {} (curly braces).

number  =  10.99
my_str  =  "Hello World"
#    1st brace takes 1st value, 2nd brace takes 2nd value.
print ( " My number is {first} and my string is {second} ".format ( first = number , second = my_str )  )
#   Output:   My number is 10.99 and my string is Hello World

Section 2.2: Applying f-strings

f-strings does exactly the same job as format( ) method , but f-string comes with much easier syntax. They're new in Python 3.

  • Variable name is defined inside {} (curly braces).

  • Put f letter before the string, in this case compiler knows that you're going to use f-string format.

  • No space allowed between f and ".  E.x: f"{hello}"

number  =  10.99
my_str  =  "Hello World"
#    Don't even bother about different data types
print ( f" My number is {number} and my string is {my_str} "  )
#   Output:   My number is 10.99 and my string is Hello World
