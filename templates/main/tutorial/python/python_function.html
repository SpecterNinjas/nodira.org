Python Function



Section 1.1: Python Function Overview

Function is a set of organized and reusable actions that will perform specific operation once it's called.

Python already provides built-in functions such as input ( ) , print ( ) , len ( ) and so on. However, you can also create your own functions (defined as def).

  • Note that the body content of your function must be indented, otherwise error raises.

def function_name ( parameters ) :
    #   some logic goes here

Section 2.1: Defining Simple Functions

  • Defining magic function. This function will not print anything, unless it's called :

def magic ( ) :
    print ( " Hello World! " )

  • Calling magic function :

magic ( )
#   Output:  Hello World!

Section 2.2: Defining Arguments Within Functions

  • Defining magic function with an argument inside the parentheses.

def magic ( name ) :
    print ( " Hello , " + name )

  • Calling magic function with an argument:

magic ( " Selena Gomez! ")
#   Output:  Hello , Selena Gomez!
magic ( " Alex! ")
#   Output:  Hello , Alex!

  • We also can do some complex logic using if/else statements:

def is_even ( number ) :
    if number  %  2  ==  0:
        print ( str ( number )  +  "   is  even  number. " ) #    Don't forget to change data type to 'str' , otherwise Error occurs!
    else :
        print ( str ( number )  +  "   is  odd  number. " )

  • Calling is_even function with an argument:

is_even ( 5 )
#   Output:  5  is  odd  number.
is_even ( 10 )
#   Output:  10  is  even  number.
is_even ( )
#   Error: No argument given.
is_even ( 20 , 23 )
#   Error: Only 1 argument required.

Section 2.3: Defining Multiple Arguments Within Functions

  • Defining func function with 2 arguments inside the parentheses.

def func ( country , capital ) :
    print ( capital +  "  is  a  capital  of   "  + country )

  • Calling func function with 2 arguments:

func ( "England" , "London" )
#   Output:  London  is  a  capital  of  England
func ( "Russia" , "Moscow" )
#   Output:  Moscow  is  a  capital  of  Russia
func ( "Italy" )
#   Error :   2 arguments required.

Section 3.1: Arbitrary Arguments ( *args ) Within Functions

  • Defining nums function with arbitrary number of arguments inside the parentheses.

def nums ( *args ) :
    print ( list ( args ) )

  • Calling nums function with arbitrary arguments:

nums ( [ 1 , 2 , 3 , 4 , 5 ] )
#   Output:   [ 1 , 2 , 3 , 4 , 5 ]

Section 3.2: Keyword Arbitrary Arguments ( **kwargs ) Within Functions

**kwargs stands for Keyword Arbitrary Arguments.

def nums ( **kwargs ) :
    for  name , cost   in   kwargs.items ( ) :
         print ( name , cost )

  • Calling nums function with keyword arbitrary arguments:

nums ( num1  =  1 , num2  =  2 , num3  =  3 )
#   Output:
num1  =  1
num2  =  2
num3  =  3

nums ( { 'one'  =  1 , 'two'  =  2 , 'three'  =  3 } )
#   Output:
one  =  1
two  =  2
three  =  3

Section 4.1: Pass Statement

If defined function has no body (content), it'll throw an Error, in order to avoid this error pass statement is used.

def magic ( ) :
    pass

print ( magic ( ) )
#   Output:   None

Section 4.2: Return Statement

  • return statement will just return the value, once we call the function it'll show nothing. Because, it'll just return the value.

  • In order to show the value (output), we must use print function.

  • We also can do some complex logic using if/else statements:

def is_even ( number ) :
    if number  %  2  ==  0:
        return str ( number )  +  "   is  even  number. " #    Don't forget to change data type to 'str' , otherwise Error occurs!
    else :
        return str ( number )  +  "   is  odd  number. "

  • Calling is_even function with an argument:

is_even ( 5 )   #   Output: Nothing, because we function will just return the value
print ( is_even ( 7 ) )   #   Output:  7  is  odd  number.
print ( is_even ( 10 ) )   #   Output:  10  is  even  number.
print ( is_even ( ) )   #   Error: No argument given.

Section 4.3: Default (Optional) Parameter in Functions

  • Default values are initialized inside the function parentheses.

  • If function is called without any arguments, then default values will be displayed.

  • If function is called with arguments, then default values will be overridden.

def magic ( default_val = " World! " ) :
    print ( " Hello + default_val " )

magic ( " Selena Gomez! ") # Default value will be overridden
#   Output:  Hello Selena Gomez!
magic ( )
#   Output:  Hello World!
