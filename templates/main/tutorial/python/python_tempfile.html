Python Tempfile



Section 1.1: Python Tempfile Overview

Python tempfile module is intended to create temporary files and directories. TemporaryFile , NamedTemporaryFile , TemporaryDirectory

In this section, we'll learn more about how to create temporary file and how to handle temporary directories on a deep level.

Section 2.1: Python TemporaryFile Syntax

  • The syntax of TemporaryFile ( ) function:

TemporaryFile ( mode = 'w+b' , buffering = None , encoding = None , newline = None , suffix = None , prefix = None ,
dir = None , errors = None )

Parameter Description
mode mode to open ﬁle, default = w + b.
buffering Store data in memory temporarily.
encoding Encryption & Decryption of a file.
newline Adding a newline, default = None.
suffix One or more letters added at the end of a word to modify the word's meaning.
prefix One or more letters added to the beginning of a word to modify its meaning
dir Dirname to place tempfile, default = None.
errors A failure to complete a task, default = None

#    TemporaryFile
import     tempfile

print ( " Temporary File example . . . " )
temp  =  tempfile.TemporaryFile ( )

try :
    print ( " TemporaryFile Reference : " , temp )
    print ( " TemporaryFile Name : " , temp.name )
finally :
    temp.close ( )

#   Output:
Temporary File example . . .
TemporaryFile Reference :   _io.BufferedRandom name = 7
TemporaryFile Name :  7



Section 2.2: Python NamedTemporaryFile Syntax

  • The syntax of NamedTemporaryFile ( ) function:

NamedTemporaryFile ( mode = 'w+b' , buffering = None , encoding = None , newline = None , suffix = None , prefix = None ,
dir = None , delete = True , errors = None )

Parameter Description
mode mode to open ﬁle, default = w + b.
buffering Store data in memory temporarily.
encoding Encryption & Decryption of a file.
newline Adding a newline, default = None.
suffix One or more letters added at the end of a word to modify the word's meaning.
prefix One or more letters added to the beginning of a word to modify its meaning
dir Dirname to place tempfile, default = None.
delete To delete ﬁle on closure, default = True
errors A failure to complete a task, default = None

#   NamedTemporaryFile
import     tempfile

with  tempfile.NamedTemporaryFile ( delete  =  False )   as   t:
    t.write ( "Python tempfile is cool" )
    path  = t.name
    print   path

with   open ( path )   as   t:
    print   t.read ( )

#   Output:
/tmp/tmp6pireJ
Python tempfile is cool
