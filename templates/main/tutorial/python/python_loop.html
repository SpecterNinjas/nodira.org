Python Loops



Section 1.1: Python Loops Overview

Like in all other programming languages, loops play very important role in programming. Python provides 2 categories of loop and in this section, we'll cover both: for and while loop.

In python, for loop will take care of indexing itself, whereas C/C++ , Java programmers don't have this functionality.

Section 2.1: Python For Loop

For loops are commonly used to access the items of lists, tuples, sets and dictionaries individually.

for i in ['a', 'b', 'c'] :
     print ( i )      #    Output:    a b c

In the above mentioned example, for loop will iterate each item of a list one by one.

Let's see 2 more examples:

  • range(4) function returns a list of numbers, starting from 0 till 4 (not included) : [ 0 , 1 , 2 , 3 ]

for i in range(4) :
     print ( i )      #    Output:    0 1 2 3

  • In this example, for loop will iterate over the string characters one by one:

for i in "nodira" :
     print ( i )      #    Output:    n o d i r a

Section 2.2: For Loop - Break Statement

break statement will stop the loop.

for  x   in  [ 'a', 'b', 'c' , 'd' ] :
     print ( x )
     if x  ==  'c' :
        break#    Loop breaks once given condition becomes True

  • Once for loop reaches to the item c , condition becomes True and loop stops functioning.

#    Output:
a
b

Section 2.3: For Loop - Continue Statement

continue statement will skip the item and continues accessing from the next item.

for  x   in  [ 'a', 'b', 'c' , 'd' ] :
     if x  ==  'c' :
        continue#    skip current the element
     print ( x )

  • Once for loop reaches to the item c, condition becomes True and loop will skip the item.

#    Output:
a
b
d

Section 3.1: Python While Loop

While loops are used to access a set of items as long as specified condition becomes True

i = 0
while i < 4 :
    print ( i )      #    Output:    0 1 2 3
    i += 1     #    Important   

  • It's important to increment i , otherwise it remains being 1 and will loop forever.

Section 3.2: While Loop - Break Statement

break statement will stop the loop.

i = 0
while i < 4 :
    if i == 3:
        break
    print ( i )      #    Output:    0 1 2
    i += 1     #    Important   

  • Once while loop reaches to 3 , condition becomes True and loop stops functioning.

#    Output:
0
1
2

Section 3.3: While Loop - Continue Statement

continue statement will skip the item and continues accessing from the next item.

i = 0
while i < 5 :
    if i == 3:
        continue
    print ( i )      #    Output:    0 1 2 4
    i += 1     #    Important   

  • Once while loop reaches to 3, condition becomes True and loop will skip 3.

#    Output:
0
1
2
4
