Python Find & Filter Function



Section 1.1: Python Find Overview

The find ( ) function is used to search certain value/expression over string and returns its index.

Section 1.2: Python Find Syntax

  • The syntax of find ( ) function:

find ( value , start , end )

Parameter Description
value Required parameter for find function. Searches/Finds certain value over string.
start Starting point for searching. (Optional)
end Ending point for searching. (Optional)
Section 1.3: Find Function in Practice

  • If searching value isn't found find (  ) function returns -1 , otherwise returns the index of 1 st occurrence of a value

my_str  =   "Python Tutorial at Nodira courses"
#   returns the index of 1 st occurrence 'T'
print ( my_str.find ( "T" ) ) #   7
print ( my_str.find ( "Nodira" ) ) #   19

  • Starting and ending parameters are mostly used to find out 2 nd and 3 rd occurrences of a pattern

  • Let's do some practice with starting and ending parameters:

my_str  =   "Python Tutorial is cool!"
#   returns the index of 2 nd occurrence 'o'
print ( my_str.find ( "o" , 5 ) ) #   10
#   starting & ending parameter
print ( my_str.find ( "t" , 5 , 10 ) ) #   9
#   If pattern isn't found, function returns -1
print ( my_str.find ( "Nodira" , 5 ) ) #   -1


Section 2.1: Python Filter Overview

The filter ( ) function is used to filter the items of an iterable over certain criteria.

Section 2.2: Filter Function Syntax

  • The syntax of filter ( ) function:

filter ( function , iterable )

Parameter Description
function Required parameter to filter the items of an iterables over certain criteria.
iterable Iterable in which function executes a set of criteria.
Section 2.3: Filter Function Practice

  • We used lambda inline function to define our function. Learn More

names  =    [  "John" , "Alexa" , "Abraham" , "Lincoln" ]
#   Using  inline  lambda  function
list ( filter ( lambda x : len ( x ) > 5 , names ) ) #   [ 'Abraham' , 'Lincoln' ]
list ( filter ( lambda x : len ( x ) < 5 , names ) ) #   [ 'John' , 'Alexa' ]

  • In this example, we used user defined function to define our criteria. Learn More

names  =    [  "John" , "Alexa" , "Abraham" , "Lincoln" ]
def longer ( name ) :
    return   len ( name ) > 5
#   Filters and shows the result, where criteria fully satisfied
list ( filter ( longer , names ) ) #   [ 'Abraham' , 'Lincoln' ]
