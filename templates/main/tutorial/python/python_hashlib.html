Python Hashlib



Section 1.1: Hashlib Overview

Hashlib module provides secure hash and message digest algorithms.

Hashlib  provides FIPS secure hash algorithms SHA1, SHA224, SHA256, SHA384, and SHA512 and RSAâ€™s MD5 algorithm.

Section 2.1: FIPS Hash Algorithm

FIPS hashing are used to provide secure message encryption.

FIPS hashing provides constructors to create each hashing objects: sha1( ) , sha224( ) , sha256( ) , sha384( ) and sha512( ).

import   hashlib
security  =  hashlib.sha256 (  )
security.update ( b"Secure text!" )
security.digest (  ) #   Output:
b'\x15\x0ee\xf4\xabne\xf9\xbe\x8f\x95\xb2\xa5\x94\xdd\x94\xdf\xd2\xe9%\x9bn'
security.hexdigest (  ) #   Output:
b'\x15\x0ee\xf4\xabne\xf9\xbe\x8f\x95\xb2\xa5\x94\xdd\x94\xdf\xd2\xe9%\x9bn'
security.digest_size #   32
security.block_size #   64

Section 2.2: RSA - md5( ) Algorithm

RSA MD5 algorithm ( defined in Internet RFC 1321 ).

RSA hashing provides constructors to create each hashing objects: md5( ) .

import   hashlib
security  =  hashlib.md5 (  )
security.update ( b"Secure text!" )
security.digest (  ) #   b'\x89\x10(\xd7#R\x81]cw\xc8\x01&Z6\xc3'
security.hexdigest (  ) #   891028d72352815d6377c801265a36c3
security.digest_size #   16
security.block_size #   64
