Python Projects For Practice

Topic : To-Do List

Description : A list of the tasks that you have to do, or things that you want to do. Must contain basic functions like : add, delete and edit.

Topic : Quiz Web Application

Description : Short Competitive Game in which a user will answer to the questions. Must check & count correct and wrong answers.

Topic : Weather Web Application

Description : Simple Weather forecast app using API. User can view the weather in any location and option to choose Celsius or Fahrenheit.

Topic : Simple Calculator

Description : Calculator must have simple UI/UX design and performs basic calculations such as addition, subtraction, multiplication, and division.

Topic : Web Crawling

Description : A web crawler copies webpages by the search engine. The web crawler validates links, HTML code and extracts data from the website.

Topic : URL Shortener

Description : Shorten Long URLs easily using is.gd, v.gd, bit.ly, shorte.st, TinyURL services to easily share them via social media using your app.

Topic : Tic-Tac-Toe Game

Description : In this game, players take turns putting marks on a board. Who first gets 3 marks in a row (vertical, horizontal and diagonal) wins, and the other side loses the game.

Topic : Guessing Number

Description : One side thinks of a secret number while the 2nd player tries to guess the number. After each guess, the 1st player answers either "Higher", "Lower" or "Correct".

Topic : Simple Chatbot

Description : Simple Interactive Chatbot that is used to implement basic command line interactions with user. Mainly, used in live contact chats for receiving basic information about user.

Topic : Rock, Paper, Scissors Game

Description : Creating a simple Rock, Paper, Scissor Game to use basic Python functions in practice. Use Math.random function to randomly provide one of the options.
