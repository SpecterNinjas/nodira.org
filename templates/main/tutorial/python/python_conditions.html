Python Conditions



Section 1.1: Python Conditions

Basically, python uses if, elif and else statements to create conditions.

Section 2.1: If Statement

  • The body of if statement needs to be indented.

age = 10
if age < 18:
    print ("You are still kid.") #    Don't forget indentation

  • Error raises when you forget to indent the body of if statement.

  • In general, many programming languages (C/C++, Java) uses curly braces { } to define its contents. But, python breaks this golden rule and uses indentation instead.

age = 10
if age < 18:
print ("You are still kid.") #    IndentationError: expected an indented block

Section 2.2: Elif Statement

  • The body of elif statement also needs to be indented.

  • Some programming languages like (C/C++, Java) call this statement as else if statement. But, in python this has been renamed as elif statement.

age = 10
if age < 18:
    print ("You are still kid.") #    Don't forget indentation
elif age == 18:
    print ("You'r exactly 18 years old.") #    Don't forget indentation

Section 2.3: Else Statement

  • The body of else statement also needs to be indented.

  • else statement doesn't accepts any condition. It'll get the opposite condition of if statement.

age = 10
if age < 18:
    print ("You are still kid.") #    Don't forget indentation
else:#    age >= 18
    print ("You are mature, that's cool:)") #    Don't forget indentation

age = 10
if age < 18:
    print ("You are still kid.") #    Don't forget indentation
elif age == 18:
    print ("You'r exactly 18 years old.") #    Don't forget indentation
else:#    age > 18
    print ("You are mature, that's cool:)") #    Don't forget indentation

Section 2.4: Nested if/else statements

  • Nested if/else statements - if condition is given under certain condition. Simply, if you provide if/else statement inside another if/else statement, this means you have used so called Nested if/else conditions.

age = 15
if age < 18:
    if age > 10:
        print ("You're between 10 and 18") #    Don't forget indentation
    else:
        print ("Your age is less than 10")
else:#    age >= 18
    if age < 50:
        print ("You're between 18 and 50")
    else:
        print ("Your age 50+")

Section 3.1: Inline Conditional Statements

  • Inline Conditional Statement - widely used in many languages. This feature can only be used when the body of conditional statements consists of a single line code.

  • if the body of if, elif or else statements only consist of 1 line of code, then you don't need indentation and their condition needs to added next to them.

age = 10
if age < 18:     print ("You are still kid.")
elif age == 18:     print ("You'r exactly 18 years old.")
else:     print ("You are mature, that's cool:)")

Section 4.1: The Ternary Expression

  • The Ternary Expression - like Inline Conditional Statements can only be used when the body of conditional statements consists of a single line code.

  • The syntax of Ternary Expression might seem a little bit weird, but can be really helpful.

    Syntax : condition_if_true if condition else condition_if_false

age = 10
print ("You are still kid")   if age < 18   else  print ("You are mature, that's cool:)")

  • As we said, syntax might be weird and sometimes confusing. So let's start one by one.

#   if statement
age = 10
print ("You are still kid.")if age < 18

#   if/else statement
age = 10
print ("Less than 10")if age < 18else print ("Higher or equal to 10")

  • The if/else statements can also be written as nested ternary operators, but make sure that the body of if/else statement consists of a single line of code.

  • Intentionally, we didn't provide the syntax of nested ternary operators, because it is too long and more confusing. It's much better and easy to understand the syntax in practise.

#   nested ternary operator
a = 10
print ("Less than 18")if a < 18else  print ("Between 18 and 50")if a < 50else   print ("Higher than 50")

Detailed step by step explanation of nested ternary operators:

  1. Checks if a is less than 18, simply prints "Less than 18"

  2. if a is higher or equal to 18, it'll start checking 2 conditions

  3. if a is less than 50, prints "Between 18 and 50"

  4. if a is higher or equal to 50, prints "Higher than 50"
