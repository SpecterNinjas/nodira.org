Python Array



Section 1.1: Python Array Overview

An array is a data structure that stores values of same data type. In Python, this is the main difference between arrays and lists.

While python lists can contain values corresponding to different data types, arrays in python can only contain values corresponding to same data type. In this section, we will understand the Python arrays with few examples.

from  array   import   *

  • Once we defined our library for using array, here is syntax:

array_name  =  array ( parameter ,  iterable )

Parameter Description
b Represents signed integer of size 1 byte
B Represents unsigned integer of size 1 byte
c Represents character of size 1 byte
u Represents unicode character of size 2 bytes
w Represents unicode character of size 4 bytes
l Represents signed integer of size 4 bytes
L Represents unsigned integer of size 4 bytes
i Represents signed integer of size 2 bytes
I Represents unsigned integer of size 2 bytes
f Represents floating point of size 4 bytes
d Represents floating point of size 8 bytes
h Represents signed integer of size 2 bytes
H Represents unsigned integer of size 2 bytes
Section 2.1: Accessing Array Elements

  • Note that in arrays indexing starts with 0.

  • my_arr[0] returns 1st element of an array, while my_arr[-1] returns last element.

  • len(my_arr) returns total number of elements of an array.

from   array   import   *#    importing array library
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )#    ' i ' :  Represents signed integer of size 2 bytes
print ( my_arr [ 0 ] ) #    1
print ( my_arr [ 1 ] ) #    2
print ( my_arr [ -1 ] ) #    3
print ( len ( my_arr ) #    3

  • Accessing array elements using loop:

from   array   import   *#    importing array library
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )#    ' i ' :  Represents signed integer of size 2 bytes
for   x   in   my_arr :
    print ( my_arr [ x ] ) #    Output:   1  2  3

Section 3.1: Appending Value To Array

  • With the help of append( ), you can add any value :

from   array   import   *#    importing array library
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )#    ' i ' :  Represents signed integer of size 2 bytes
my_arr.append(6)
print ( my_arr ) #    1 , 2 , 3 , 6

  • With the help of insert ( index , value ), you can insert value on specified index:

from   array   import   *#    importing array library
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )#    ' i ' :  Represents signed integer of size 2 bytes
my_arr.insert ( 1 , 5 ) #    inserts 5 to the index of 2
print ( my_arr ) #    1 , 5 , 2 , 3

  • With the help of extend ( my_arr2 ), you can merge 2 array elements.

from   array   import   *
my_arr1  =  array ( 'i' ,  [ 1 , 2 , 3 ] )
my_arr2  =  array ( 'i' ,  [ 4 , 5 , 6 ] )
my_arr1.extend ( my_arr2 ) #    Merges 2 arrays
print ( my_arr1 ) #    1 , 2 , 3 , 4 , 5 , 6

  • With the help of fromlist ( my_list ), you can merge list elements to the array.

from   array   import   *
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )
my_list  = [ 4 , 5 , 6 ]
my_arr.fromlist ( my_list ) #    Merges list elements to array
print ( my_arr ) #    1 , 2 , 3 , 4 , 5 , 6

Section 2.2: Removing Array Elements

  • With the help of remove ( value ), you can remove specified value array.

from   array   import   *
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )
my_arr.remove(2) #    removes 2 from the array
print ( my_arr ) #    1 , 3

  • pop ( ) removes last element of an array.

from   array   import   *
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )
my_arr.pop( ) #    removes last element of an array
print ( my_arr ) #    1 , 2

Section 3.1: Applying index( ) method

  • index ( value ) returns the index position of a specified value from the array.

from   array   import   *
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )
my_arr.index(  3  ) #    returns the index position of 3
print ( my_arr ) #    2

Section 3.2: Reversing The Order Of Array Elements

  • reverse( ) function will reverse the order of array elements.

from   array   import   *
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )
my_arr.reverse( ) #   reverses the order of array elements
print ( my_arr ) #    3 , 2 , 1

Section 3.3: Finding The Occurrence Of Array Elements

  • count ( value ) function will count how many times specified element occurred in an array.

from   array   import   *
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 , 3 , 2 , 3 , 3 ] )
#   count how many times 3 occurred in an array
print ( my_arr.count(  3  ) ) #    4

Section 3.4: Converting Array To List

  • tolist ( ) function will convert an array to a list.

from   array   import   *
my_arr  =  array ( 'i' ,  [ 1 , 2 , 3 ] )
my_list  =  my_arr.tolist( ) #   converts array to list
print ( type ( my_list ) ) #   Data type :  ' list '

Section 4.1: Accessing Multidimensional Arrays

my_list  =  [ [ 1 , 2 , 3 ] , [ 2 , 3 , 3 ] , [ 4 , 2 , 3 ] ]
print ( my_list [ 0 ] )   #     [ 1 , 2 , 3 ]
print ( my_list [ 1 ] )   #     [ 2 , 3 , 3 ]
print ( my_list [ 2 ] )   #     [ 4 , 2 , 3 ]

  • Accessing certain elements of a list :

my_list  =  [ [ 1 , 2 , 3 ] , [ 2 , 3 , 3 ] , [ 4 , 2 , 3 ] ]
print ( my_list [ 0 ]  [ 0 ] )   #     1
print ( my_list [ 1 ]  [ 2 ] )   #     3
print ( my_list [ 2 ]  [ 1 ] )   #     2
