Python Print Function



Section 1.1: Print Function Overview

print (  ) function is used to print something on a screen.

Section 1.2: Python Print Syntax

  • The syntax of print( ) function:

print ( anything , sep , end , file , flush )

Parameter Description
anything Literally, anything you want to print on a screen. (Required)
sep sep (separator) will separate objects and be placed between 2 objects. If it's not defined by user, by default empty space is used. (Optional)
end end is declared to define how your output should end. separate objects and be placed between 2 objects. If it's not defined by user, by default endline is used. (Optional)
file File object with write function. (Optional)
flush True/False. If True, output will be flushed out, if it's False, output will be buffered. (Optional)
Section 1.3: Print Function Practice

  • Note that string and numerical data types can't be simply combined, Error occurs.

print ( "Hello, Python!" )#   Hello, Python!
a  = 10
b  = 99.9
print ( a )#   10
print ( b )#   99.9

  • Passing arguments with commas:

a  = 10
b  = 99.9
c  = "Hello, Python!"
print ( a , b , c )#   10   99.9    Hello, Python!

  • Passing arguments with plus ( '+' ) :

c  = "What's up buddy?"
print ( "Hi, " + c )#   Hi, What's up buddy?

Note that you can't simply merge 'str' with numerical data type. You have 2 options:
1 . Pass them with commas
2 . Change numerical data type to 'str' with str ( )

a  = 10
b  = "Hello, Python!"
print ( a + b)#   TypeError :   unsupported operand type for 'int' and 'str'
print ( a , b)#   10 Hello, Python!
print ( str ( a ) + b)#   10 Hello, Python!

  • print ( ) function will calculate the content first:

print ( 10 + 10.1 )#   20.1
print ( "3" + "4" )#   34
print ( "a" + "b" )#   ab

Section 2.1: Print Function Parameters

  • sep (separator) will separate objects and be placed between 2 objects. If it's not defined by user, by default empty space is used:

print ( "Alexa" , "Justin" , "John" , sep  = " - " ) #    Alexa - Justin - John
print ( "Alexa" , "Justin" , "John" , sep  = " : " ) #    Alexa : Justin : John

  • end is declared to define how your output should end. separate objects and be placed between 2 objects. If it's not defined by user, by default endline is used.

print ( "Alexa" , "Justin" , "John" , sep  = " - " , end  = " . . . " ) #    Alexa - Justin - John . . .
print ( "Alexa" , "Justin" , "John" , sep  = " : " , end  = " The End " ) #    Alexa : Justin : John The End
