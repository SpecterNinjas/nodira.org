Python Copying Data



Section 1.1: Python Copy Overview

The copy ( ) function simply copies the data of one iterable to another iterable.

In Python, copying data is divided into 2 categories: Shallow Copy and Deep Copy

Section 1.2: Difference between Shallow Copy and Deep Copy

Shallow Copy - a method that copies data collection without bothering of its elements.

Deep Copy - a method that copies data collection with its elements, mostly used to copy nested elements.

  • The syntax of shallow copy function:

import copy
copy.copy ( iterable )

  • The syntax of deep copy function:

import copy
copy.deepcopy ( iterable )

Section 2.1: Shallow Copy in Practice

  • Note that copied data will not be updated if you add some data to the original dataset.

import copy
names  =    [  "Alex" , "Justin" , "John" ]

#   performing shallow copy
copied_names  =  copy.copy ( names )
print ( names ) #   [ 'Alex' , 'Justin' , 'John' ]
print ( copied_names ) #   [ 'Alex' , 'Justin' , 'John' ]

#   appending some data
names.append( "Selena" )
print ( names ) #   [ 'Alex' , 'Justin' , 'John' , 'Selena' ]
print ( copied_names ) #   [ 'Alex' , 'Justin' , 'John' ]

import copy
names  =    [  [ 0 , 1 ] , [ 2 , 3 ] ,   [ 4 , 5 ] ]

#   performing shallow copy
copied_names = copy.copy ( names )
print ( names ) #   [ [ 0 , 1 ] , [ 2 , 3 ] , [ 4 , 5 ] ]
print ( copied_names ) #   [ [ 0 , 1 ] , [ 2 , 3 ] , [ 4 , 5 ] ]

#   changing data
names [ 1 ] [ 1 ]  =  100
print ( names ) #   [ [ 0 , 1 ] , [ 2 , 100 ] , [ 4 , 5 ] ]
print ( copied_names ) #   [ [ 0 , 1 ] , [ 2 , 100 ] , [ 4 , 5 ] ]

  • Shallow Copy - a method that copies data collection without bothering of its elements.

Section 2.2: Deep Copy in Practice

  • Note that copied data will not be updated if you add some data to the original dataset.

import copy
names  =    [  [ 0 , 1 ] , [ 2 , 3 ] ,   [ 4 , 5 ] ]

#   performing deep copy
copied_names = copy.deepcopy ( names )
print ( names ) #   [ [ 0 , 1 ] , [ 2 , 3 ] , [ 4 , 5 ] ]
print ( copied_names ) #   [ [ 0 , 1 ] , [ 2 , 3 ] , [ 4 , 5 ] ]

#   changing data
names [ 1 ] [ 1 ]  =  100
print ( names ) #   [ [ 0 , 1 ] , [ 2 , 100 ] , [ 4 , 5 ] ]
print ( copied_names ) #   [ [ 0 , 1 ] , [ 2 , 3 ] , [ 4 , 5 ] ]

  • Deep Copy - a method that copies data collection with its elements, mostly used to copy nested elements.

Section 3.1: Slicing Method For Shallow Copy

  • Note that slicing method will function the same as shallow copy:

names  =    [  "Alex" , "Justin" , "John" ]
#   performing shallow copy using slicing
copied_names  = names [ : ]
#   appending some data
names.append( "Selena" )
print ( names ) #   [ 'Alex' , 'Justin' , 'John' , 'Selena' ]
print ( copied_names ) #   [ 'Alex' , 'Justin' , 'John' ]
