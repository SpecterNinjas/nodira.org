Python String Manipulation



Section 1.1: String Manipulation Overview

This section focuses on strings and applies several functions over them.

Let's some practice and learn about fundamental functions on a deep level.

Section 2.1: Changing String Capitalization

Python provides 6 types of Strings Capitalization :

  • str.upper (  ) - method converts all lowercase characters in a string into uppercase characters.

  • str.lower (  ) - method converts all uppercase characters in a string into lowercase characters.

  • str.capitalize (  ) - first letter capitalized and all other characters will be lowercased.

  • str.title (  ) - converts first character of each word to uppercase and remaining to lowercase.

  • str.swapcase (  ) - converts uppercase to lowercase and lowercase to uppercase.

  • str.casefold (  ) - returns a string where all characters are lowercase and used to implement caseless string matching.

"this IS an Upper string".upper (  ) #    THIS IS AN UPPER STRING
"THis IS a LOWER string".lower (  ) #    this is a lower string
"this IS a STRING".capitalize (  ) #    This is a string
"this IS a string".title (  ) #    This Is A String
"this IS a STRING".swapcase (  ) #    THIS is A string
"this IS a XßΣ".casefold (  ) #    this is a xssσ

Section 2.2: concatenating variables

Python provides 2 types of method for concatenating variables :

  • str.format(  )

  • f - strings

a , b , c  =  10 , 10.5 ,"Python"

#    str.format
"5 + 5 will be : {}". format ( a )#    5 + 5 will be : 10
"5 + 5 will be : {} , not {}". format ( a , b )#    5 + 5 will be : 10 , not 10.5
"5 + 5 will be : {answer} , not {wrong}". format ( answer  =  a , wrong  =  b )#    5 + 5 will be : 10 , not 10.5

#    f-string
f"5 + 5 will be : {a}" #    5 + 5 will be : 10
f"{c} is cool!" #    Python is cool!

Section 2.3: character accessing

Python does not support char data type. Therefore, stings are a combination chars representing in unicode format.

brackets [  ] are used for character accessing.

Minus Indexing will start counting from backside.  E.x: [ -1 ] is last element.

a  =  "Python"
#    character accessing
a [ 0 ] #    P
a [ 1 ] #    y
a [ -1 ] #    n
a [ -2 ] #    o

#    slicing   [ from : to ]
a [ 0 : 2 ] #    Py
a [ 2 : 4 ] #    th
a [ 1 : 5 ] #    ytho

Section 2.4: String Module

The string module provides a collection of constants.

Python provides 9 types of constants :

  • string.ascii_letters - Combination of the ascii_lowercase and ascii_uppercase constants.

  • string.ascii_lowercase - All ASCII lowercase letters

  • string.ascii_uppercase - All ASCII uppercase letters

  • string.digits - All Digits 0-9

  • string.hexdigits - All Hexadecimal Digits

  • string.octdigits - All Octal Digits 0-7

  • string.punctuation - All ASCII characters which are considered punctuation characters

  • string.printable - All ASCII characters which are considered printable

  • string.whitespace - All ASCII characters which are considered whitespace

import    string
string.ascii_letters #    abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ
string.ascii_lowercase #    abcdefghijklmnopqrstuvwxyz
string.ascii_uppercase #    ABCDEFGHIJKLMNOPQRSTUVWXYZ
string.digits #    0123456789
string.hexdigits #    0123456789abcdefABCDEF
string.octdigits #    01234567
string.punctuation #    !"#$%&\'()*+,-./:;<=>?@[\\]^_`{|}~
string.printable #     All ASCII characters which are considered printable
string.whitespace #     \t\n\r\x0b\x0c

Section 2.5: String Length

Python provides len (  ) function to measure length of a string and all iterables ( like list , tuple , set ).

a  =  len ( "Python!" )#     counts number of characters
print ( a )#    7

nums  =  len ( [ 1 , 2 , 19 , 34 , 22 ] ) #     counts number of items in a list
print ( nums ) #    5

chars  =  len ( { "a" , 2 , 19 , "b" } ) #     counts number of items in a set
print ( chars ) #    4

Section 2.6: strip ( ) function

Python provides 3 methods to remove whitespace of a string from left and right sides :

  • str.strip (  ) - method removes whitespace from left and right sides.

  • str.lstrip (  ) - method removes whitespace from left side.

  • str.rstrip (  ) - method removes whitespace from right side.

a  =  "        Python is cool!        "
a.strip ( ) #     removes whitespace from left and right sides
print ( a ) #    "Python is cool!"

a.lstrip ( ) #     removes whitespace from left sides
print ( a ) #    "Python is cool!        "

a.rstrip ( ) #     removes whitespace from right sides
print ( a ) #    "        Python is cool!"

Section 2.7: split ( ) function

The split ( sep  = None ) function takes a string and returns it by dividing into substrings :

a  =  "Python is cool!".split (  )
print ( a )#     [ 'Python' , 'is' , 'cool!' ]

b  =  "Selena, Alex, Justin".split ( ',' )
print ( b )#     [ 'Selena' , 'Alex' , 'Justin' ]

Section 2.8: join ( ) function

The join (  ) function takes a list of substrings and returns a string :

" ".join ( [ "Python" , "is" ,"cool!" ] ) #     Python is cool!

"<-->".join ( [ "Python" , "is" ,"cool!" ] ) #     Python<-->is<-->cool!

"***".join ( [ "Python" , "is" ,"cool!" ] ) #     Python***is***cool!

Section 2.9: in & not in

in keyword checks a substring existence , while not in checks not existence of a substring :

"o" in "Python"#     True
"a" in "Python"#     False

"Py" not in "Python"#     False
"a" not in "Python"#     True

Section 3.2: Replace (  ) Function

Python provides replace ( old , new ) function to replace old substring with new ones.

"Learn more about String".replace ( "String" , "Numbers" )#     Learn more about Numbers
