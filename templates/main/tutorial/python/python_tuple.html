Python Tuple



Section 1.1: Python Data Type

In this section, we'll cover one of these: Python Tuple

  • Tuple - a collection of ordered elements.

  • Tuple - immutable (meaning that we can't change its elements).

  • Tuple - we can't add or delete any element. Remains fixed.

  • Tuple - supports indexing. Elements start with an index of 0, not 1.

  • Tuple - parentheses( ) are used to define list data type.

Section 1.2: Tuple Example

Tuple elements are separated by commas and closed by parentheses.

a  =  ( 1 , 'dog' , 10.99 , 'cat' , 0.678 )# defining simple tuple
print ( a )# 1 , 'dog' , 10.99 , 'cat' , 0.678

b  =  ( 10 , 23.99 , 10.99 , 0.56 , 0.99 )# defining numerical tuple
print ( b )# 10 , 23.99 , 10.99 , 0.56 , 0.99

a  =  ( 'puppy' , 'dog' , 'hen' , 'cat' , 'cow' )# defining string tuple
print ( a )# 'puppy' , 'dog' , 'hen' , 'cat' , 'cow'

Section 1.3: Accessing Tuple items

Tuple items can be accessed through index (like lists). E.x: tuple [index]

tuple [ start : stop : step ]

a  =  ( 1 , 'dog' , 10.99 , 'cat' , 0.678 )# index 0 is 1st element of a tuple
print ( a[0] )#    1

  • First element starts with index 0, not 1.

b  =  ( 10 , 23.99 , 10.99 , 0.56 , 0.99 )# show items with index 0 , 1 , and 2.
print ( b[0:3] )# 10 , 23.99 , 10.99

  • Show items starting with index 0 till 3 (not included).

a  =  ( 'puppy' , 'dog' , 'hen' , 'cat' , 'cow' )# show items with index 2 and 3.
print ( a[2:4] )# 'hen' , 'cat'

  • Show items starting with index 2 till 4 (not included).

a  =  ( 'puppy' , 'dog' , 'hen' , 'cat' , 'cow' )# -index means start counting from backside
print ( a[-1] )# 'cow' -> 1st index from backside

  • minus( - ) means start from backside.

  • E.x: [-1] -> last element, [-2] -> second element from the back.

a = ( 'puppy' , 'dog' , 'hen' , 'cat' , 'cow' )# [a:] means start from index 'a' till the end.
print ( a[1:] )# 'dog' , 'hen' , 'cat' , 'cow'

a  =  ( 'puppy' , 'dog' , 'hen' , 'cat' , 'cow' )# show items with index 2 and 3.
print ( a[2:4] )# 'hen' , 'cat'

Section 1.4: Combining 2 tuples

  • merged_tuple = tuple1 + tuple2

a  =  ( 1 , 'dog' , 10.99 , 'cat' , 0.678 )
b  =  ( 10 , 23.99 , 10.99 , 0.56 , 0.99 )
c  =  a + b#   merging 2 tuples
print ( c )#   1 , 'dog' , 10.99 , 'cat' , 0.678 , 10 , 23.99 , 10.99 , 0.56 , 0.99

Section 1.5: 'step' - 3rd argument of a tuple

b  =  ( 10 , 23.99 , 10.99 , 0.56 , 0.99 )
print ( b[ : : 2 ] )#   10 , 10.99 , 0.99
print ( b[ : : 3 ] )#   10 , 0.56

Section1.6: Reversing a tuple

b  =  ( 10 , 23.99 , 10.99 , 0.56 , 0.99 )
print ( b[ : : -1 ] )#    0.99 , 0.56 , 10.99 , 23.99 , 10  ( reverse step )

  • We must remember that we can't change , add and delete items of a tuple. But, we can reverse or sort them.

Section 1.7: Changing Tuple Item - Not Allowed

a  =  ( 'puppy' , 'dog' , 'hen' , 'cat' , 'cow' )
a [ 2 ]  =  'sheep'#    Tuple items can't be changed or deleted.
print ( a )#    Error

Section 1.8: Checking item Existence

tuple1  =  ( 'puppy' , 'dog' , 'cat' )
print ( 'hen'  in   tuple1)#    False
print ( 'hen'  not  in   tuple1)#    True

Section 1.9: Built-in functions

a  =  ( 'puppy' , 'dog' , 'hen' )
len ( a )#   3   (displays total number of items)

  • len (  ) shows total number of items in a tuple

a  =  ( 'puppy' , 'dog' , 'hen' )
del a #    Deleting whole tuple
print ( a )#    Error

  • del removes whole tuple

a  =  ( 'a' , 'b' , 'c' , 'a' , 'b' , 'a' )
a.count ( 'a' )#   3   (total number of 'a' in a tuple)
a.count ( 'b' )#   2   (total number of 'b' in a tuple)
a.count ( 'c' )#   1   (total number of 'c' in a tuple)

  • count (  ) returns total number of a specified item in a tuple.

a  =  ( 'a' , 'b' , 'c' , 'a' , 'b' , 'a' )
a.index ( 'a' )#   0   (returns index of 1st 'a')
a.count ( 'b' )#   1   (returns index of 1st 'b')
a.count ( 'c' )#   2   (returns index of 1st 'c')

  • index (  ) returns the index of 1st matched occurrence of a specified item.

Section 2.1: Removing dublicated items

We can simply change data type of tuple to set and then convert it again to tuple.

a  =  ( 10 , 23.99 , 23.99 , 10.99 , 10, 0.56 , 0.56 , 0.99 )
a  =  tuple ( set ( a ) )#    0.56 , 0.99 , 10 , 10.99 , 23.99

  • set (  ) automatically removes dublicated items.
