Section 2.1: Python Concentration

Let's apply mathematical operators over different data types string and numerical :

#   '+' sign combines 2 strings
a  =  "Python"
b  =  " programming"
print ( a + b ) #   Python programming

#   '+' sign add 2 numerical numbers
a  =  3
b  =  4
print ( a + b ) #   7

#    '*' sign repeats strings
a  =  "apple" * 3 #   appleappleapple
b  =  "a" * 4 #   aaaa
c  =  "a" + 4 #   TypeError:  cannot concatenate 'str' and 'int'

#    '*' sign multiplies numbers
a  =  5 * 3 #   15
b  =  2.5 * 4 #   10.0
c  =  "a" + str ( 4 ) #   a4 

Section 2.2: Python Switch Variables

It's super easy to switch variables in Python :

a  =  5
b  =  6
#     switching variables
a , b  =  b , a
print ( a ) #   6

Section 2.4: Difference Between is and ==

When you'r new in Python, it might be very confusing for beginners to distinguish the difference of is and == :

  • x  ==  y will compare only equality the values of x and y.

  • x  is  y will compare whether x and y refer to the same object or not.

#     comparing is and == operators
a , b  =  5 , 6
x  =  "Python is cool!"
y  =  "Python is cool!"

print ( a  ==  b ) #   False
print ( a  is  b ) #   False

print ( x  ==  y ) #   True
print ( x  is  y ) #   True

#     comparing is and == operators
a  =  [ 3 , 4 , 5 ]
b  =  a.copy (  ) #    shallow copy
c  =  a #    c referencing to a

print ( a  ==  b ) #   True
print ( a  is  b ) #   False

print ( a  ==  c ) #   True
print ( a  is  c ) #   True
