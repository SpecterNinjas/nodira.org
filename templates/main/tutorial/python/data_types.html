Python Data Types



Section 1.1: Python Data Types

Basically, Python variables will differentiate with respect to the data that they will store.

Fortunately, Python already built for us 7 categories of data types to use.

Text Type Numeric Type Sequence Type Mapping Type Set Type Boolean Type Binary Type
str int list dict set bool bytes
float tuple frozenset bytearray
complex range memoryview
Section 1.2: Getting Data Types using type() function

  • type ( ) function returns data type of a specified variable

a  =  2
print ( type ( a ) )    # Output: <'int'>
b  =  9223372036854775807
print ( type ( b ) )    # Output: <'int'>

pi  =  3.14
print ( type ( pi ) )    # Output: <'float'>

c  =  'A'
print ( type ( c ) )    # Output: <'str'>
name  =  'John Doe'
print ( type ( name ) )    # Output: <'str'>

q  =  True
print ( type ( q ) )    # Output: <'bool'>

q  =  None
print ( type ( q ) )    # Output: <'NoneType'>

Section 1.4: Creating variables and assigning values

To create a variable in Python, all you need to do is specify the variable name, and then assign a value to it.

Python uses = to assign values to variables. There's no need to declare a variable beforehand, Python will take care of everything. There's no way to declare a variable without assigning it an initial value.

Integer Floating point String Boolean Null
a = 2 a = 2.14 a = 'ABC' a = True a = None
b = 10 b = 10.9 b = '102ABC' b = False b = None
c = 199 c = 199.01 c = 'Roman Collak' c = True c = None

Variable always needs to be from left to right. So the following will give you an syntax error.

0 = x         #    SyntaxError : can't assign to literal

Rules for variable naming:

  1. Variables names must start with a letter or an underscore.

    2. x = True        # valid
    _y = True     # valid
    9x = False    # starts with numeral
    $y = False    # starts with symbol

  2. The remainder of your variable name may consist of letters, numbers and underscores.

    3. has_0_in_it  =  "Still Valid"

  3. Names are case sensitive.

    4. x = 9
    y = X*5
    >>>NameError: name 'X' is not defined


Assigning multiple variables:

a , b , c  =  1 , 2 , 3
print ( a , b , c ) # Output: 1 2 3

a , b , c  =  1 , 2
print ( a , b , c ) # ValueError: need more than 2 values to pack

a , b  =  1 , 2 , 3
print ( a , b , c ) # ValueError: too many values to unpack

  • Underscore (_) for assigning unwanted values:

a , b , _  =  1 , 2 , 3
print ( a , b ) # Output: 1 2

a , b , _  =  1 , 2 , 3 , 4
print ( a , b ) # ValueError too many values to unpack

  • Assigning a single value to several variables simultaneously:

a  =  b  =  c  =  1
print ( a , b , c ) # Output:    1 1 1

  • Now, let's see a bit difficult example:

a  =  b  =  c  =  1 # all three names a, b and c refer to same int object with value 1
print ( a , b , c ) # Output:    1 1 1
b  =  2# b now refers to another int object, one with a value of 2
print ( a , b , c ) # Output:    1 2 1

  • Another example about over assigning:

a  =  1
print ( a ) # Output: 1
a  =  " New value over written "
print ( a , b , c ) # Output:    New value over written

Unlike, other programming languages Python will take care of their data types. Awesome!

Section 1.5: Python Data Types in details

  • Integer Number

    int : integer number
    a  =  1
    b  =  21
    c  =  1234
    d  =  2334215323

  • Floating Point Number

    float : floating point number
    a  =  1.2
    b  =  21.13
    c  =  1234.422
    d  =  2334215323.01

    In some programming languages (C/C++ , Java), it is also referred as double.

  • Boolean Values

    bool : Boolean value
    a  =  True
    b  =  False

    Boolean value can be only True or False.

  • String

    str : String
    a  =  'ABC'
    b  =  'Azam'
    c  =  b'Azam' #   Assigning string in Binary format

  • List

    list : Collection of ordered items (Can be reassigned)
    a  =  [ 1 , 2 , 3 ]
    b  =  [ 'A' , 2 , 'CD', ( 1 , 2 ) ]

  • Tuple

    tuple : Collection of ordered items (Can't be reassigned)
    a  =  ( 1 , 2 , 3 )
    b  =  ( 'A' , 2 , 'CD' , [ 1 , 2 ] )

  • Set

    set : Unordered collection of Unique items
    a  =  { 1 , 2 , 3 }
    b  =  { 'A' , 2 , 'CD' , [ 1 , 2 ] }

  • Dictionary

    dict : Unordered collection of unique key-value pairs
    a  =  { k  =  'Hello World' , d  =  'ABC' , b  =  'ABC123' }
