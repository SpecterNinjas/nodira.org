Python Lambda



Section 1.1: Python Lambda Overview

The lambda creates an inline anonymous ( without name) function. In general, normal user-defined function is created using def keyword, where function name is required. However, anonymous functions so called lambda function has no name, that's why we call them anonymous.

Section 1.2: Lambda Function Syntax

lambda arguments : expression

The lambda might have multiple arguments and only one expression. When function is invoked this expression is going to be output.

Section 2.1: Difference between def and lambda functions

  • Let's assume we have normal user defined function called magic:

def magic ( ) :
    return " Python is cool! "

print ( magic ( ) )   #   Python is cool!

  • The translation of this function in lambda will be:

hey  = lambda : " Python is cool! "

print ( hey ( ) )   #   Python is cool!

Section 2.2: Arguments with lambda functions

  • Let's now convert def function with arguments:

def   power ( a ) :
    return   a  **  2

print ( power ( 8 ) )   #   64

  • The translation of this function in lambda will be:

power  =  lambda   a  :  a  **  2

print ( power ( 8 ) )   #   64

Section 2.3: Multiple Arguments with lambda functions

add  =  lambda   a , b :  a  +  b

print ( add ( 7 , 8 ) )   #   15
print ( add ( 7) )   #   Error: 2 arguments required, given 1.
print ( add ( ) )   #   Error: 2 arguments required, given 0.

Section 3.1: Why we need lambda functions ?

With the help of lambda , we can do simple tasks quickly.
For exampe, Python provide several useful functions like reduce , map , filter , search and so on.

Now, let's see some of the examples using lambda with these functions.

It's totally fine if you don't 100% understand what's going on, because we'll cover more in-depth understanding about these functions later on.

my_list  =    [ 1  ,  2  ,  -3  ,  7  ,  5  ,  -8 ]
list ( filter ( lambda x : x > 0 , my_list ) )
  #   Output:  [ 1 , 2 , 7 , 5 ]
list ( map ( lambda x : abs ( x ) , my_list ) )
  #   Output:  [ 1 , 2 , 3 , 7 , 5 , 8 ]

  • As you see, it's so useful to use lambda inline function, rather than normal def function.
