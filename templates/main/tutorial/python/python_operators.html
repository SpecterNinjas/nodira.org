Python Operators



Section 1.1: Python Operator Categories

In this chapter, we'll cover python operators.
Basically, python operators are divided into 3 categories: Arithmetic , Logical/Bitwise and Boolean.

Section 2.1: Python - Arithmetic Operator

Let's start with Arithmetic operators.

Operator Syntax Description
Addition a + b Finds sum of 2 numbers.
Subtraction a - b Finds subtraction of 2 numbers.
Division a / b Finds division of 2 numbers.
Floor Division a // b Finds floor division of 2 numbers.
Multiplication a * b Finds multiplication of 2 numbers.
Power a ** b Finds power of a number.
Modulus a % b Finds modulus of 2 numbers.
Logarithm log (a, base) Finds logarithm of a number.
Trigonometry sin( ) , cos( ) , tan( ) ... Performs trigonometric operations.
Section 2.1.1: Arithmetic Operator - Addition

a , b = 3 , 4     #    Addition
a + b      #    Output:    7

import operator
a , b = 3 , 4     #    Built-in Function for Addition
operator.add(a,b)     #    Output:    7

Section 2.1.2: Arithmetic Operator - Subtraction

a , b = 3 , 4     #    Subtraction
a - b      #    Output:    -1

import operator
a , b = 3 , 4     #    Built-in Function for Subtraction
operator.sub(a,b)     #    Output:    -1

Section 2.1.3: Arithmetic Operator - Division

a , b = 3 , 4     #    Division
a / b      #    Output:    0.75

  • In Python 3.x and Python 2.x division differs.

import operator
a , b = 3 , 4     #    Built-in Function for Division
operator.div(a,b)     #    Output:    0.75

Section 2.1.4: Arithmetic Operator - Floor Division

a , b = 14 , 3     #    Floor Division
a // b      #    Output:    4

  • Floor Division - returns the “floor” of the result, or the result of the division rounded down to the nearest integer.

import operator
a , b = 14 , 3     #    Built-in Function for Floor Division
operator.floordiv(a,b)     #    Output:    4

Section 2.1.5: Arithmetic Operator - Multiplication

a , b = 3 , 4     #    Multiplication
a * b      #    Output:    12

import operator
a , b = 3 , 4     #    Built-in Function for Multiplication
operator.mul(a,b)     #    Output:    12

Section 2.1.6: Arithmetic Operator - Power

a , b = 3 , 4     #    Power
a ** b      #    Output:    81

  • Multiplication can be found using 2 libraries: operator and math.

  • The only difference is operator.pow( ) returns integer,while math.pow( ) returns float.

import operator
a , b = 3 , 4     #    Built-in Function for Power
operator.pow(a,b)     #    Output:    81

import math
a , b = 3 , 4     #    Built-in Function for Power
operator.pow(a,b)     #    Output:    81.0

Section 2.1.7: Arithmetic Operator - Modulus

  • Modulus (remainder) can be found using % sign.

a , b = 3 , 4     #    Modulus
a % b      #    Output:    3

  • Note: If numerator < denominator, modulus(remainder) = numerator.

a , b = 10 , 5     #    If numerator < denominator, modulus = numerator.
a % b      #    Output:    0

import operator
a , b = 3 , 4     #    Built-in Function for Modulus
operator.mod(a,b)     #    Output:    3
operator.mod(10,5)     #    Output:    0

Section 2.1.8: Arithmetic Operator - Logarithm

  • Logarithm can be found using : math

  • log(a , base=e). If 2nd argument is omitted, then by default base will be equal to natural e.

import math
a = 3     #    Built-in Function for Logarithm
math.log(a)     #    Output:    1.0986122886681098         (by default, base = e)
math.log(math.e)     #    Output:    1.0         (by default, base = e)
math.log(27,3)     #    Output:    3.0         (given that base = 3)

Section 2.1.9: Arithmetic Operator - Trigonometry

import math
math.sin(5)     #    Output:    -0.9589242746631385
math.cos(math.pi)     #    Output:    -1

Section 3.1: Python - Logical Operator

Logical operators includes : and, or, not.

Logical Operator Syntax Description
and a and b if both conditions are true, returns True, otherwise False.
or a or b if one of conditions is true, returns True, returns False if both of them are false.
not not a if condition is true, returns False, if it is false, returns True.

  • and: if both conditions are true, returns True, otherwise False.

#    Let's suppose we've 2 conditions: a and b.
print(a = True and b = True)     #    True
print(a = True and b = False)     #    False
print(a = False and b = False)     #    False

  • or: if one of conditions is true, returns True, returns False if both of them are false.

#    Let's suppose we've 2 conditions: a and b.
print(a = True and b = True)     #    True
print(a = True and b = False)     #    True
print(a = False and b = False)     #    False

  • not: if condition is true, returns False, if it is false, returns True.

a = True
print(a)     #    False

a = False
print(a)     #    True

Section 4.1: Python - Bitwise Operator

Bitwise operations alter binary strings of every bit.
First, we need to convert it into two's complement notation which consists of 8 bits.
Then, we can start performing bitwise operands on every bit.

Bitwise Symbol Description
~ a Bitwise NOT
a & b Bitwise AND
a | b Bitwise OR
a ^ b Bitwise XOR
>> a Bitwise Right Shift
<< a Bitwise Left Shift
Section 5.1: Python - Comparison Operator
Operator Syntax Description
> a > b returns True if a is bigger than b, otherwise returns False.
< a < b returns True if a is less than b, otherwise returns False.
<> a <> b returns True if a is not equal to b, otherwise returns False.
!= a != b The same as <> (not equal) operator.
>= a >= b returns True if a is bigger or equal to b, otherwise returns False.
<= a <= b returns True if a is less or equal to b, otherwise returns False.
== a == b returns True if a is equal to b, otherwise returns False.
