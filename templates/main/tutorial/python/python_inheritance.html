Python Inheritance



Section 1.1: Python Inheritance Overview

This section focuses on Inheritance method and how to implement in Python projects.

Inheritance is a feature that allows a class to make a specific section of your project's code reusable with the possibility of adding or removing certain features later.

Let's do some practice and learn about inheritance on a deep level.

Section 2.1: Python Inheritance Practice

Let's create parent and child class that uses inheritance for code reusability :

class   Figure :#     defining Parent class
    def   __init__ ( self , width , height ) :
        self.width  =  width
        self.height  =  height

    def   product ( self ) :#     defining a simple method
        print ( self.width * self.height )

class   Rectangle ( Figure ) :#     inheriting Parent class
    pass#     inherits all features of Parent Class

figure  =  Figure ( 5 , 6 )
rectangle  =  Rectangle (3 , 4 )

print ( figure.width )#      5
print ( figure.height ) #      6

print ( rectangle.width )#      3
print ( rectangle.height ) #      4

Section 3.1: Inheritance Conclusion

Inheritance is the ability to to inherit the properties of another class.

Advantages of Inheritance

  • It supports DRY ( Don't Repeat Yourself ) method making your code ( classes, methods ) reusable.

  • Reusable methods between parent class and child classes.

  • Less memory storage and optimization of the execution time.

  • Repetition of the code is reduced & minimized so that it results to code consistency and cleanliness.

Disadvantages of Inheritance

  • Hard to evolve and maintain complex class hierarchies.

  • It results in a very tight binding between a superclass and its subclasses.

  • Inheritance relationships generally can't be altered at runtime.
