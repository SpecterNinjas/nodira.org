Python Attribute Access



Section 1.1: Attribute Access Overview

An Attribute Access is a term that's mainly used in terms of class.

Class attributes are variables of a class that are shared between all of its instances.

In Python, Dot Notation and Set/Get Functions & Properties used to access the attribute of any class.

Section 2.1: Python Dot Notation

Let's create a simple class called Person and do some practice:

class  Person (  ) :

#   Defining simple class 'Person'
    def  __init__ ( self , name , surname ) :
        self.name  =  name
        self.surname  =  surname

#   Creating some instances of a class
mike  = Person ( name  = "Mike" , surname  = "Ross" )
harvey  = Person ( name  = "Harvey" , surname  = "Specter" )

Now , let's access name and surname of instances using Dot Notation:

print ( mike.name ) #   Mike
print ( mike.surname ) #   Ross



Section 2.2: Set/Get Functions Overview

An Attribute Access can be also achieved by defining Set & Get Functions.

Let's add these functions to our class Person :

class  Person (  ) :

#   Defining simple class 'Person'
    def  __init__ ( self , name , surname ) :
        self.name  =  name
        self.surname  =  surname

#   Defining set function
    def  set ( self , rename ) :
        self.name  =  rename

#   Defining get function
    def  get ( self ) :
        print ( f" Name: {self.name} and Surname: {self.surname} "  )

#   Creating some instances of a class
mike  = Person ( name  = "Mike" , surname  = "Ross" )

Now , let's access class attributes using set and get functions :

print ( mike.name ) #   Mike
mike.set ( "Mikeee" ) #   setting name
mike.get (  ) #   Name: Mikeee and Surname: Ross



Section 2.3: Property Overview

An Attribute Access can be also achieved by defining property.

The syntax of property :

@ Attribute_Name.setter


@radius.setter is a setter function that stores the value in self.__name.    Executes every time when value is set.

class  Person (  ) :

#   Defining simple class 'Person'
    def  __init__ ( self , name ) :
        self.name  =  name

#   Defining property
    @property
    def  name ( self ) :
        return  self.__name

#   applying property to set a name
    @name.setter
    def  name ( self , name ) :
        self.__name  =  name
        print ( "Success!" )

#   Creating some instances of a class
mike  = Person ( name  = "Mike" )

Now , let's access name of instances using property:

print ( mike.name )
#   Success!
#   Mike

mike.name  =  "Harvey"#   Rename attribute
print ( mike.name )
#   Success!
#   Harvey
