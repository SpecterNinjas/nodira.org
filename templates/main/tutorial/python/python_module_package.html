The Difference between Module and Package



Section 1.1: Python Module Overview

Python Module - is a single python file containing a set of functions and algorithms that you will execute in your software application. Modules are importable.

Section 1.2: Creating Python Module

To create Python Module, you simply need save your source code file with an extension of .py

Section 1.3: Python Module Practice

  • We'll create 2 python files : module.py and main_page.py

#   module.py
def magic ( ) :
    print ( " Python is cool! " )

#   main_page.py
import module#   only module name without .py extension
module.magic ( ) #   Python is cool!



Section 2.1: Python Package Overview

Python Package - is a folder of multiple Python modules or can also be referred as a library.

Section 2.2: Creating Python Package

An example of Python Package tree (Folder name --> "python_package") :

  • __init__.py

  • module1.py

  • module2.py

Remember that imported modules needs to be declared in __init__.py file.

Section 2.3: Python Package Practice

  • We'll create 2 python files : module.py and main_page.py

#   __init__.py
import module1#   only module name without .py extension
import module2
module1.magic ( ) #   Python is cool!
module2.hello ( ) #   Hello, Python!

#   module1.py
def magic ( ) :
    print ( " Python is cool! " )

#   module2.py
def hello ( ) :
    print ( " Hello, Python! " )
