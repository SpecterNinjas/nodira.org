Python Descriptor



Section 1.1: Python Descriptor Overview

Python provides 3 types of methods to define descriptors. They're: __get__ , __set__ , __delete__ methods.

In this section, we'll learn more about how to create and use these methods on a deep level.

Section 1.2: Python Descriptor Syntax

  • The syntax of descriptor methods:

descriptor.__get__ ( self , obj , type  =  None ) #    returns 'value'
descriptor.__set__ ( self , obj , value ) #    returns 'None'
descriptor.__delete__ ( self , obj ) #    returns 'None'

Section 2.1: Applying Descriptors in Practice

  • Note that Data descriptors are defined as objects in both __get__ and __set__, whereas non-data descriptors only define a __get__ method.

#   Simple usage of Descriptor
class   Car ( object ) :

    def   __init__ ( self , brand = 'BMW'  ) :
        self.brand  = brand

    def   __get__ ( self , obj , type = None ) :
        return    f" My Car's Brand {self.brand} "

    def   __set__ ( self , obj , brand ) :
        if   type ( brand )  ==  'str' :
            self.brand  =  brand
        else  :
            print ( " Brand name must be only string! " )

#   Accessing Descriptor:
class   Desc ( object ) :
    name  = Car ( )

x  = Desc ( )
x.brand  =  "Ferrari"
print ( x.brand ) #   Ferrari

Section 2.2: Applying Descriptors in Practice

  • Another simple, but very good explanation of defining a descriptor.

#   Simple usage of descriptor
class   Person ( object ) :

    name  =  'Alex'

    def   __get__ ( self , obj , type = None ) :
        print ( " Getting Person name . . . "  )
        return    self.name

    def   __set__ ( self , obj , pname ) :
        print ( " Setting Person name: "  , pname )
        self.name = pname

    def   __delete__ ( self , obj ) :
        print ( " Deleting Person name . . . "   )
        del  self.name

#   Accessing Descriptor:
class   Name ( object ) :
    t  = Person ( )

desc  = Name ( )
print ( desc.t )
#    Getting Person name...
#   Alex

desc.t  = 'John'
#   Setting Person name: John

del  desc.t
#    Deleting Person name...
