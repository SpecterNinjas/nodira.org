Python OS Path



Section 1.1: Python os.path Overview

os.path is one of the most widely used modules in Python. Because, literally, it contains loads of very fundamental and useful functions on pathnames. And there're only 2 ways of passing parameters to it, either strings or bytes.

In this section, we'll learn about os.path and about its most useful functions on a deep level.

Section 2.1: os.path.exists

os.path.exists function is used to check the existence of a path. To avoid an error, it's recommended to use under try/except block.

import   os.path
print ( os.path.exists ( "/nodira/python/tutorial" ) )
#    Output:   True / False based on the existence of a path.

Section 2.2: os.path.isdir

os.path.isdir function is used to check whether it's a directory or not. To avoid an error, it's recommended to use under try/except block.

import   os.path
print ( os.path.isdir ( "/nodira/python/home" ) )
#    Output:   True / False based on the link of a directory.

Section 2.3: os.path.isfile

os.path.isfile function is used to check whether it's a file or not. To avoid an error, it's wise to use under try/except block.

import   os.path
print ( os.path.isfile ( "/nodira/python/index.txt" ) )
#    Output:   True / False based on the existence of a file.

Section 2.4: os.path.ismount

os.path.ismount function is used to check whether it's a mount point or not. What's a mount point ?

import   os.path
print ( os.path.ismount ( "/nodira" ) )
#    Output:   True / False based on the existence of a file.

Section 3.1: os.path.join

os.path.join function is used to combine 2 paths.

import   os.path
path  =  '/nodira/python/'
print ( os.path.join ( path ,"index.py" ) )
#    Output:   /nodira/python/index.py

Section 3.2: os.path.basename

os.path.basename function is used to return basename (filename) from given path.

import   os.path
path  =  '/nodira/python/index.py'
print ( os.path.basename ( path ))
#    Output:   index.py

Section 3.3: os.path.dirname

os.path.dirname function is used to return the path of a file.

import   os.path
path  =  '/nodira/python/index.py'
print ( os.path.dirname ( path ))
#    Output:   /nodira/python/

Section 3.4: os.path.split

os.path.split function will separate the path and file .

import   os.path
path  =  '/nodira/python/index.py'
print ( os.path.split ( path ))
#    Output:   ( ' /nodira/python/ ' , ' index.py ' )

Section 4.1: os.getcwd

os.getcwd function will return the current working directory .

import   os.path
print ( os.getcwd ( ))
#    Output:   ( ' /nodira/python/index.py ' )

Section 4.2: os.path.abspath

os.path.abspath function will return the absolute path of a file .

import   os.path
print ( os.path.abspath ( " index.py " ))
#    Output:   ( ' /nodira/python/index.py ' )

Section 4.3: os.path.isabs

os.path.isabs function will check whether given path is absolute or not .

import   os.path
print ( os.path.isabs ( " /nodira/python " ))
#    Output:   True
