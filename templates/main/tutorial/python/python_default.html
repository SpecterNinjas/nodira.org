Section 1.1: Python Course Overview

Python is more popular than ever, and is being used everywhere from back-end web servers, to front-end game development, and everything in between. Python is a true general purpose language and is quickly becoming a must-have tool in the arsenal of any self-respecting programmer.

In this section, we'll learn about Python general concepts & usage on a deep level.

Section 2.1: Python Programming - General Concepts

Let's learn some of the general concepts of Python :

  • Open Source and Community Development - all modern versions of Python is absolutely free of charge and open source. Importantly, according to the statistics of IEEE Spectrum ranking Python is #1 community support programming language and the #1 "Most Wanted" and #2 "Most Loved" language, according to StackOverflow's 2019 Survey.
    This means that no matter what problem you're trying to solve, chances are there are already strong people working on a solution. Chances are also good that they have shared code, documentation, tutorials, and examples to help program a solution in Python.

  • User friendly Data Structures - Python has built-in list and dictionary data structures which can be used to construct fast runtime data structures. Further, Python also provides the option of dynamic high-level data typing which reduces the length of support code that is needed.

  • Productivity and Speed - Python has clean object-oriented design, provides enhanced process control capabilities, and possesses strong integration and text processing capabilities and its own unit testing framework, all of which contribute to the increase in its speed and productivity. Python is considered a viable option for building complex multi-protocol network applications.

Section 2.4: Python Conclusion

Python is an easy to learn, powerful programming language. It has efficient high-level data structures and a simple but effective approach to object-oriented programming. Python’s elegant syntax and dynamic typing, together with its interpreted nature, make it an ideal language for scripting and rapid application development in many areas on most platforms.

Advantages of Python

  • Easy to learn & understand fundamental concepts. The most demanded language for productivity.

  • Python is OOP driven, offers good support for objects, modules, and other re-usability mechanisms.

  • Python is a robust programming language and provides an easy usage of the code lines.

  • Python provides large standard libraries that include the areas like string operations, Internet, web service tools, operating system interfaces and protocols.

Disadvantages of Python

  • Python is quite slower in comparison with C / C ++ and Java.

  • Python isn't the best for memory intensive tasks as it's an interpreted language.

  • Python's database access layer is found to be bit underdeveloped and primitive.

  • Python is evolving continuously, with constant evolution there is little substantial documentation available for the language.
