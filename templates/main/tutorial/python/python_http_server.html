Python Http Server



Section 1.1: Http Server Overview

The Http Server is a module that defines classes for implementing HTTP servers (Web servers).

http.server is not recommended for production. It only implements basic security checks.

Let's do some exercises and learn about Http Server on a deep level.

Section 1.2: Running Http Server

Let's run our server on a default port :

python  -m   http.server #   server is running on port: 8000

Let's run our server at port 9000 :

python  -m   http.server   9000

Section 1.3: Serving files

Let's say we have 3 files : main.py , forms.py and server.py

Once we run our Http Server by following example:

#   server.py
import   http.server
import   socketserver #   used to setup a network serve

PORT  =  8000

handler  =  http.server.SimpleHTTPRequestHandler
httpd  =  socketserver.TCPServer ( ( "" , PORT ) , handler )
print ( "Serving Port " , PORT )
httpd.serve_forever (  )

The SimpleHTTPRequestHandler class of the SimpleHTTPServer module allows the files at the current directory to be served.

Section 2.1: GET, POST, PUT Methods

Http Server handling methods - GET, POST, PUT :

from    http.server   import   BaseHTTPRequestHandler ,   HTTPServer

class   HandleRequests ( BaseHTTPRequestHandler ) :

    def   header_info ( self ) :
        self.send_response ( 200 )
        self.send_header ( 'Content-type' , 'text/html' )
        self.end_headers (   )

    def   GET_Func ( self ) :
        self.header_info (   )
        self.wfile.write ( " GET Method Done! " )

    def   POST_Func ( self ) :
        self.header_info (   )
        content_len  =  int ( self.headers.getheader ( 'content-length' , 0 ) )
        post_body = self.rfile.read ( content_len )
        self.wfile.write ( f"POST Request: {post_body} " )

    def   PUT_Func ( self ) :
        self.POST_Func (  )

host  =  ' '
port  =  80
HTTPServer ( ( host , port ) , HandleRequests ).server_forever ( )
