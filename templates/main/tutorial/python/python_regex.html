Python Regular Expressions



Section 1.1: Regular Expression Overview

Regular Expression is one of the widely used and most popular module for searching a specific pattern.

Basically, RegEx can be used to check specific pattern in a string or to validate form fields.

In this section, we'll learn more about how to create and use Regular Expressions on a deep level.

Section 2.1: Match( ) Function

match( ) function is used to match the beginning of a string.

  • The syntax of match( ) function:

match ( pattern , text )

import  re #   importing RegEx module
text  =   "1234Python"
pattern  =   "123"
match  = re.match ( pattern , text )

if   match:
    print ( "Found"  )
else :
    print ( "Not Found"  )

#   Output:  Found

Section 2.2: Search( ) Function

search( ) function is used to match the beginning of a string.

  • The syntax of search( ) function:

search ( pattern , text )

import  re #   importing RegEx module
text  =   "Python is so cool"
pattern  =   "is so"
match  = re.search ( pattern , text )

if   match:
    print ( "Found"  )
else :
    print ( "Not Found"  )

#   Output:  Found

Section 2.3: Flags Keyword

search( ) function offers 4 flags for searching on a deep level:

Flags

re.IGNORECASE

re.MULTILINE

re.DOTALL

re.DEBUG

Description

Ignores the case of a character

Makes ^ match the begin of a line and $ the end of a line

Match everything including newlines

Turn on DEBUG Mode

text  =  re.search ( "p" , "Python" ) #   No Match
text  =  re.search ( "p" , "Python" , flags  =  re.IGNORECASE ) #   Match
text  =  re.search ( "p" , "Python" , flags  =  re.DEBUG ) #   Turning on DEBUG Mode

Section 3.1: Metacharacters in RegEx

Python provides 10 metacharacters to define Regular Expressions.

Character Description Example
^ Starting with ^a
$ Ending with a$
. Any Single Character (Except Newline) a..n
+ One more occurrence a+
* Zero more occurrence a*
| OR a|b
\ To Escape Special Characters \b
[ ] Range of characters [ a - z ]
{ } Exact occurrence of a character a{2}
( ) A group of a character (a|b)c
