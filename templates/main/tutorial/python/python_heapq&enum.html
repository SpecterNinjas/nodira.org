Python Heapq & Count Function



Section 1.1: Python Heapq Overview

The heapq( ) library is mainly used to find out n - largest and n - smallest items from a collection. Basically, python makes it easy for programmers by providing special library for finding them, whereas it will take at least 20 lines of code for other programming languages (C/C++, Java) to find n - largest and n - smallest item of a collection.

heapq.nlargest ( integer_number , iterable )

heapq.nsmallest ( integer_number , iterable )

Parameter Description
integer_number Integer number that defines how many to take. (Required)
iterable Iterable or a list of numbers. (Required)
Section 1.2: Finding Largest And Smallest Element

  • nlargest ( how_many, numbers )  will return a list of specified n - largest elements of a collection :

import   heapq  #    importing heapq library
my_list  =  [ 10 , 22 , 13 , 92 , -3 , 3 , 34 , 2 , 53 ]
print ( heapq.nlargest (4, my_list ) ) #    [ 92 , 53 , 34 , 22 ]

  • nsmallest ( how_many, numbers )  will return a list of specified n - smallest elements of a collection :

import   heapq  #    importing heapq library
my_list  =  [ 10 , 22 , 13 , 92 , -3 , 3 , 34 , 2 , 53 ]
print ( heapq.nsmallest (3, my_list ) ) #    [ -3 , 2 , 3 ]

Section 1.3: Heapify

  • heapify ( numbers )  will return a list of numbers, where 1st element will be the smallest :

import   heapq  #    importing heapq library
my_list  =  [ 10 , 22 , 13 , 92 , -3 , 3 , 34 , 2 , 53 ]
heapq.heapify ( my_list ) #    [ -3 , 2 , 3 , 10 , 22 , 13 , 34 , 92 , 53 ]
print ( my_list [ 0 ]) #    -3



Section 2.1: Python Count Overview

In general, count( ) method is used to find frequency of an element in a collection. Frequency means how many times an element occurred in a collection. During practice, you'll obtain more deep understanding of Count method and where to apply.

  • Simply, it'll count how many times elements occurred in a collection.

from   collections   import   Counter  #    importing Counter function
my_list  =  Counter ( [ "a" , "b" , "b" , "h" , "c" , "a" , "a" , "c" , "b" ] )
print ( my_list ) #    Counter ( { 'a' : 3 , 'b' : 3 , 'c' : 2 , 'h' : 1 } )
print ( my_list [ "b" ] ) #    3
print ( my_list [ "a" ] ) #    3
print ( my_list [ "h" ] ) #    1

Section 2.2: Finding The Occurrence - count( ) method

  • count ( value ) function will count how many times specified element occurred in lists, tuples, strings and so on.

my_nums  =    [ 1 , 2 , 3 , 3 , 2 , 3 , 3 ]
#   count how many times 3 occurred in a collection
print ( my_nums.count(  3  ) ) #    4
#   count how many times 2 occurred in a collection
print ( my_nums.count(  2  ) ) #    2

my_nums  =    [ "a" , "b" , "b" , "h" , "c" , "a" , "a" , "c" , "b" ]
print ( my_nums.count(  "a"  ) ) #    3
print ( my_nums.count(  "z"  ) ) #    0

  • Counting substring occurrence in a string

my_str  = "Awesome_Tutorials_From_Nodira"
print ( my_str.count(  "a"  ) ) #    2
print ( my_str.count(  "om"  ) ) #    2
print ( my_str.count(  "Awesome"  ) ) #    1
