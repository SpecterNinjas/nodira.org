Python Urllib



Section 1.1: Python Urllib Overview

Python urllib is one of the most widely used module for URL handling.

In this section, we'll learn more about urllib and how to handle URL path on a deep level.

>>>   pip install urllib

Section 2.1: Python URL Request

  • HTTP GET Method:

#   HTTP GET Method - used to open URL path

import   urllib.request   as   url
res  =  url.urlopen ( "https://www.apple.com/"  )
print ( res.code ) #   200
print ( res.read ( ) ) #   opens URL Webpage

Section 2.2: Python URL Parse

  • HTTP POST Method:

#   HTTP POST Method - for opening URL path

import   urllib.request   as   url
import   urllib.parse   as   parse

url_json  =  { 'username' : 'my_username' , 'password' : 'my_pass'   }
url_encode  =  parse.urlencode ( url_json ) ).encode ( 'utf-8' )
res  =  url.urlencode ( "https://secure2.store.apple.com/shop/sign_in/" )

print ( res.code ) #   200
print ( res.read ( ) ) #   opens URL Webpage

Section 2.3: Python Urllib Error

  • The urllib.error module defines the exception classes for exceptions raised by urllib.request

  • urllib.error might occur due to:

    • Wrong URL Path
    • Wrong URL Path Handling
    • Poor Internet Connection

  • Ignoring URL Error helps to debug your code:

#   Ignoring urllib.error
try :
#   some coding
except urllib.error.URLError as e: ResponseData = e.read().decode("utf8" , 'ignore')

  • Asking For A Reason URL Error :

#   Reasoning urllib.error
try :
#   some coding
except urllib.error.URLError as e: ResponseData = e.reason

Section 2.4: Python Urllib Robotparser

  • A Python module containing a single class called RobotFileParser, in which you can obtain answers regarding user agent and parameters of URL Web Page that's published in a robots.txt file.

#   urllib.robotparser

import  urllib.robotparser as   parser
rp  =  parser.RobotFileParser ( )

rp.set_url ( "http://www.musi-cal.com/robots.txt" )
rp.read ( )
req_rate  =  rp.request_rate ( "*" ) #    ( * ) means everything
print ( req_rate.requests ) #   3
print ( req_rate.seconds ) #   20
print ( rp.crawl_delay ( "*" ) ) #   6
print ( rp.can_fetch ( "*" , "http://www.musi-cal.com/cgi-bin/search?city=San+Francisco" ) ) #   False -> crawling disallowed page
print ( rp.can_fetch ( "*" , "http://www.musi-cal.com/" ) ) #   True -> crawling allowed page
