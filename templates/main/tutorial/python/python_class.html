Python Class



Section 1.1: Python Class Overview

  • Python is so well-known not only for being a scripting language, but also it supports object-oriented paradigm.

  • class keyword is used to create a class in Python.

  • In this section, we'll learn more about how to create and use a class on a deep level.

Section 1.2: Creating Simple Class

  • In this example, we'll create a simple class called Person along with a name attribute.

class   Person : #   creating class 'Person'
    name  =  'John'

  • Once we created simple class, it's time to learn how to access it:

obj  =  Person ( ) #   creating object referencing to class 'Person'
print ( obj.name ) #   John
print ( Person.name ) #   John

Section 2.1: Defining function within a Class

  • Let's define a function within a class and see simple usage of a function:

class   Person : #   creating class 'Person'
    name  =  'John'
    surname  =  'Cooper'

    def  full_name ( self ) :
        print ( f"Hello, {self.name}   {self.surname}" ) #   using f-string

  • Once we created function based class, it's time to learn how to access it:

obj  =  Person ( ) #   creating object referencing to class 'Person'

print ( obj.name ) #   John
print ( obj.surname ) #   Cooper

print ( Person.name ) #   John
print ( Person.surname ) #   Cooper

print ( obj.full_name ( ) ) #   Hello, John Cooper

Section 2.2: Defining Python Constructors

It's good practice to define constructors within a class.

Almost in all classes, we define __init__ function which will be automatically executed once class is initialized.

  • __init__ function is called initializer. This function runs first, once object referencing class is created.

  • __str__ function is sometimes called string representation. This function returns string data type.

class   Person : #   creating class 'Person'

    def  __init__ ( self , name ) : #   initializer
        self.name  =  name

    def  __str__ ( self ) : #   returns string
        return  self.name

    def  rename ( self , rename ) :
        self.name  =  rename
        print ( f" My name: { self.name } " )

  • Once we created constructor based class, it's time to learn how to access it:

alex  =  Person ( "Alex"  )
john  =  Person ( "John"  )

print ( alex.name ) #   Alex
print ( john.name ) #   John

print ( alex.__str__ ( ) ) #   Alex
print ( john.__str__ ( ) ) #   John

alex.rename ( "Amanda" )
john.rename ( "Joseph" )

print ( alex.name ) #   Amanda
print ( john.name ) #   Joseph

Section 2.3: Class Manipulation Practice

Let's do some exercise to learn about class on a deep level:

  • Defining class constructor

  • Modifying class properties

  • Deleting class properties

class   Person : #   creating class 'Person'

    def  __init__ ( self , name , surname ) : #   initializer
        self.name  =  name
        self.surname  =  surname

    def  hello ( self ) :
        print ( f" My name: { self.name } { self.surname } " )

  • Let's play with our class:

mike  =  Person ( "Mike" , "Ross"  )
harvey  =  Person ( "Harvey" , "Specter"  )

print ( mike.name ) #   Mike
print ( mike.surname ) #   Ross

print ( harvey.name ) #   Harvey
print ( harvey.surname ) #   Specter

mike.name  =  "Mikee"
mike.surname  =  "Rosamaxa"


mike.hello ( ) #   My name:  Mikee Rosamaxa
harvey.hello ( ) #   My name:  Harvey Specter

del  mike.name #    Deleting class property

  • pass  statement is used to avoid this Error.
